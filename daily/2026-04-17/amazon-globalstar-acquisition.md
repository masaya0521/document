# Amazon × Globalstar 買収：Project Kuiper の衛星接続戦略と Apple 連携

- **日付**: 2026-04-17
- **カテゴリ**: Tech
- **ソース**: [CNBC](https://www.cnbc.com/2026/04/14/amazon-globalstar-satellite-leo-internet.html) / [24/7 Wall St.](https://247wallst.com/investing/2026/04/14/amazon-rises-5-globalstar-jumps-10-as-11-57-billion-acquisition-ignites-a-satellite-war-with-spacex/) / [FinancialContent](https://markets.financialcontent.com/stocks/article/marketminute-2026-4-15-amazons-116-billion-globalstar-acquisition-the-rise-of-amazon-leo-and-the-new-satellite-monopoly)

## 概要

Amazon は 2026年4月14日、衛星通信事業者 Globalstar を約 **115.7億ドル** で買収する最終契約を締結したと発表した。Globalstar 株主は1株あたり現金 $90、または Amazon 株式 0.3210 株を選択できる。買収により Amazon の LEO（低軌道）衛星事業「Project Kuiper」（最近 **Amazon Leo** にリブランド）は、SpaceX Starlink に対抗する規模と機能を一気に獲得する見込みとなった。

Globalstar は 2022年以来 Apple の衛星通信（iPhone の Emergency SOS、Find My など）を支えてきた独占パートナーで、買収完了後は **Amazon Leo が Apple デバイスの衛星接続プロバイダー** となる。これにより Apple は現在の 24 機体制から、Amazon が 2029年までに配備予定の 3,200 機のコンステレーションを通じた広域カバレッジを利用できるようになる。

## 詳細

### 戦略的位置づけ

- **対 Starlink**: SpaceX Starlink は既に 7,000 機超を運用し衛星ブロードバンド市場を支配している。Amazon は Project Kuiper だけでは追いつけない差を、Globalstar のスペクトラム・実運用ノウハウ・顧客基盤で埋める。
- **対 AWS とのシナジー**: Amazon Leo は AWS のエッジ・Ground Station サービスと統合されることで、企業顧客向けの「クラウド⇄宇宙」一体ソリューションを描きやすくなる。
- **Direct-to-Device (D2D) への転換**: Globalstar の D2D 技術（スマホから直接衛星接続）と周波数資産は、携帯キャリア非依存の地上補完ネットワークを構築する鍵になる。

### Apple との関係

- 2022年の iPhone 14 発表以来、Apple の衛星サービスは Globalstar に全面依存してきた。
- 契約は Amazon へ移管され、Amazon Leo が Apple デバイスのキャリア的役割を継承する。
- Apple にとってはパートナーの事業体力・衛星数が大幅に増える一方、プラットフォーム（Amazon）への依存度が高まるため、長期的には独自衛星戦略・マルチキャリア化の議論が再燃する可能性がある。

### 市場インパクト

- 発表当日、Amazon 株は +5%、Globalstar 株は +10% 前後で反応。
- LEO コンステレーション市場の二大勢力化（SpaceX vs Amazon）が鮮明になり、OneWeb（Eutelsat）、AST SpaceMobile など他プレイヤーへの資本流入や再編圧力が高まる。
- 米国データセンター建設遅延のニュースと併読すると、AI / クラウド大手が「地上側の電力・部品」と「宇宙側のスペクトラム・衛星」の両方で供給制約レースに入った構図が浮かぶ。

### 規制リスク

- スペクトラム移管には FCC 承認が必要。特に Globalstar が保有する Band 53（n53）のモバイル通信用途は米国の無線政策上注目度が高い。
- 通信・宇宙・クラウドの三領域にまたがる垂直統合であり、独禁当局（FTC・DOJ）からの審査は長期化する可能性がある。

## 今後の注目点

- **クロージング時期**: FCC および各国の規制当局承認スケジュール。
- **Apple の次の一手**: 独自衛星路線（Globalstar 代替候補）を検討するか、Amazon Leo への依存をむしろ深めるか。
- **Amazon Leo の打ち上げペース**: 2029年 3,200 機目標に対する実打ち上げ進捗（ULA Vulcan、Ariane 6、Blue Origin New Glenn 等のスロット確保）。
- **Starlink 側の対応**: Starlink Direct to Cell の商用化拡大や、T-Mobile・他キャリアとの D2D 提携の深堀り。
- **他キャリア連携**: Amazon Leo が Apple 以外の OEM（Samsung、Google Pixel 等）にもホストを広げるか、逆に「Amazon アプリ／デバイス限定」に絞るか。
