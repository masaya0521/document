# OpenAI・Anthropic・Google が連携する「対中蒸留防衛」の構図

- **日付**: 2026-04-17
- **カテゴリ**: Tech
- **ソース**: [Bloomberg](https://www.bloomberg.com/news/articles/2026-04-06/openai-anthropic-google-unite-to-combat-model-copying-in-china) / [Implicator.ai](https://www.implicator.ai/openai-anthropic-google-share-attack-data-to-counter-chinese-ai-distillation/) / [Built In](https://builtin.com/articles/openai-google-anthropic-ai-model-theft-china)

## 概要

OpenAI、Anthropic、Google（Alphabet）の米フロンティア AI 3社が、中国勢による **adversarial distillation（敵対的蒸留）** を検出するための攻撃データを **Frontier Model Forum** を通じて相互共有していることが、4月初頭の Bloomberg 報道で明らかになった。これは 2023年に同フォーラムが設立されて以来、初の本格的な「運用利用」とされる。

ターゲットは DeepSeek、Moonshot AI、MiniMax といった中国の主要 AI ラボ。Anthropic は単独で **24,000 件の偽アカウントから 1,600万件の不正なリクエスト** を検出したと報告している。米中 AI 競争が「モデル性能の競争」から「**モデル知財の防衛戦**」へとフェーズ転換した象徴的な動きだ。

## 詳細

### 敵対的蒸留とは

- **手口**: フロンティアモデル（GPT-5.4、Claude Opus 4.6、Gemini 3.1 Pro 等）に対し、何百万件もの設計されたプロンプトを投げて出力を収集 → 小型モデルを fine-tune して同等の振る舞いを模倣させる。
- **コスト構造**: ゼロから学習するのに比べ桁違いに安い。中国ラボは「元モデル構築コストの数%」で「性能の 80-90%」を再現したとされる。
- **検出の難しさ**: クエリ自体は合法的な API 利用と区別しづらく、分散した偽アカウント・プロキシ経由で長期間ゆっくり収集される。

### 共有の仕組み

- **モデル**: サイバーセキュリティ企業の脅威インテリジェンス共有（ISAC モデル）に類似。1社が攻撃パターン（プロンプトのシグネチャ、行動指紋、IP/アカウント挙動）を検出すると、他社にフラグを共有。
- **対象**: アカウント挙動、プロンプト分布、レート異常、回答の保存パターンなど。実モデル重みや学習データは共有しない。
- **歴史**: Frontier Model Forum 自体は 2023年に AI 安全性協調枠組みとして設立されたが、これまで実運用での連携は限定的だった。今回が「初の防衛同盟」としての実装と評価されている。

### なぜ今この連携が成立したか

- **DeepSeek 衝撃**: DeepSeek-R1 等のオープンウェイト・モデルが安価に高性能を実現した経緯が、米国側に「蒸留対策をしないと差は急速に縮まる」との危機感を植え付けた。
- **IPO 圧力**: OpenAI（年率 250億ドル超）、Anthropic（年率 190億ドル弱）が公開市場を視野に入れる中、知財防衛は事業評価に直結する経営論点になった。
- **政府要請**: フロンティア各社は対中蒸留対策で米政府にも反トラスト面の **明示的な規制クリアランス** を求めているが、現時点では公式回答が出ていない。

### 競争と協調の二面性

- 3社は同時にモデル性能で激しく競合し続ける（2026年初に Gemini 3.1 Pro、Claude Opus 4.6/Sonnet 4.6、GPT-5.4 が立て続けにリリース）。
- 競争領域（モデル品質、価格、コンテキスト長）と防衛領域（知財・蒸留対策）を分離した「**選択的協調**」のモデルケースとなる。
- 一方で、こうした情報共有は 反トラスト法や国際協調（特に EU AI Act の透明性要件）と緊張関係を持ちうる。

## 今後の注目点

- **規制クリアランス**: 米連邦取引委員会・司法省が Frontier Model Forum 経由の情報共有に明示的な safe harbor を与えるか。
- **中国側の反応**: DeepSeek、Moonshot AI、MiniMax の対応（合法的な代替データソース、独自スケーリング、オープンソース戦略）。
- **検出技術**: 出力ウォーターマーク、プロンプトベース fingerprinting、API レイヤでの行動分析の進化。
- **オープンウェイト・モデル**: Meta（Llama）、Mistral など重みを公開する陣営は蒸留対策と相反する立場のため、米陣営内での足並みの乱れが生じうる。
- **同盟の拡張**: xAI、Microsoft、Apple（オンデバイス）など他の米国フロンティア勢が Frontier Model Forum の共有メカニズムに加わるか。
