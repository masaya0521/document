# iPS細胞再生医療の世界初実用化承認 — 20年の軌跡と今後の展望

- **日付**: 2026-02-20
- **カテゴリ**: Japan Tech
- **ソース**: [Japan Times](https://www.japantimes.co.jp/news/2026/02/19/japan/medical-product-ips-cells-ok/) / [日本経済新聞](https://www.nikkei.com/article/DGXZQOUA190360Z10C26A2000000/)

## 概要

2026年2月19日、厚生労働省の薬事審議会専門部会が、iPS細胞（人工多能性幹細胞）を用いた再生医療製品2品目の製造販売について、条件・期限付き承認（早期承認）を了承した。iPS細胞技術を用いた医薬品が実用化されるのは世界初であり、2006年の山中伸弥教授によるiPS細胞の発見から約20年を経た歴史的な到達点となる。山中教授は「社会実装に向けた大きな第一歩」とコメントした。

## 承認された2製品

### リハート（ReHeart）— 重症心不全治療

- **開発元**: クオリプス（Cuorips）— 大阪大学発ベンチャー
- **対象疾患**: 虚血性心筋症による重症心不全
- **治療法**: 他人のiPS細胞から作製した心筋細胞をシート状（直径4-5cm、厚さ0.1mm）に培養し、心臓の表面に貼り付ける
- **治験結果**: 2020〜2023年に大阪大学病院など4施設で8例の医師主導治験を実施。全員に症状改善が確認され、がん化などの重篤な有害事象はゼロ

### アムシェプリ（Amchepry）— パーキンソン病治療

- **開発元**: 住友ファーマ / Racthera
- **対象疾患**: パーキンソン病
- **治療法**: 他人のiPS細胞をドーパミン産生神経細胞の前駆細胞に分化させ、500万〜1000万個を頭部に移植
- **治験結果**: 2018〜2023年に京都大学病院で6例の臨床試験を実施。4名で運動機能の改善が確認。若年・軽症の患者で特に効果が高い傾向

## 承認制度の仕組み

今回の承認は「条件・期限付き承認制度」に基づくもので、有効性が「推定」段階でも、一定の条件の下で製造販売が認められる。承認後も最長7年間にわたり追加データを収集し、本承認の可否を専門部会が再度判断する。

この制度は2014年の医薬品医療機器等法（薬機法）改正で導入されたもので、再生医療製品の迅速な実用化を目的としている。

## 20年の開発の軌跡

| 年 | 出来事 |
|---|--------|
| 2006年 | 山中伸弥教授がマウスでiPS細胞の作製に成功 |
| 2007年 | ヒトiPS細胞の作製に成功 |
| 2012年 | 山中教授がノーベル生理学・医学賞を受賞 |
| 2014年 | 薬機法改正で条件・期限付き承認制度を導入 |
| 2014年 | 理化学研究所がiPS細胞由来の網膜シートを世界初移植 |
| 2018年 | 京都大学がパーキンソン病の臨床試験を開始 |
| 2020年 | 大阪大学が心筋シートの治験を開始 |
| 2023年 | 両製品の臨床試験が完了 |
| 2026年2月 | 厚労省専門部会が2製品の条件付き承認を了承 |

## 課題と今後の展望

### 残る課題

- **長期安全性**: iPS細胞の長期的な安全性、特にがん化リスクの監視が必要。承認後7年間のデータ収集で検証
- **コスト**: 他人由来のiPS細胞を使用することで個別作製よりコストは低減されるが、依然として高額。保険適用の範囲と薬価設定が今後の焦点
- **免疫拒絶**: 他人由来の細胞を使用するため、免疫抑制剤の併用が必要なケースがある

### 今後の注目点

- 条件付き承認後の本承認に向けた追加データ収集の進捗
- 薬価設定と保険適用の範囲決定
- 他の疾患（加齢黄斑変性、脊髄損傷、糖尿病など）への応用拡大
- 日本の再生医療産業の国際競争力への影響
- 2026年度のPMDA（医薬品医療機器総合機構）によるAI診断・モニタリングSaMDの迅速承認制度との相乗効果
