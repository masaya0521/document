# Arm AGI CPU — 35年ぶりの自社チップが AI インフラ戦争を加速

- **日付**: 2026-03-26
- **カテゴリ**: Tech
- **ソース**: [CNBC](https://www.cnbc.com/2026/03/24/arm-launches-its-own-cpu-with-meta-as-first-customer.html)

## 概要

Arm が35年の歴史で初めて自社設計・自社ブランドのデータセンター向け CPU「AGI CPU」を発表した。136コアの Neoverse V3 を搭載し、TSMC の3nm プロセスで製造される。Meta が初の顧客として契約し、OpenAI、Cloudflare、SAP など大手も顧客に名を連ねる。Arm は2031年までにこのチップで150億ドルの収益を見込んでいる。

## 詳細

### チップの仕様と性能

AGI CPU は最大136個の Neoverse V3 コアを搭載し、x86 プロセッサと比較してラックあたり2倍以上の性能を実現すると主張している。AI データセンター1ギガワットあたり最大100億ドルの設備投資削減効果があるとされる。

Arm CEO の Rene Haas はサンフランシスコでのイベントでこのチップを公開した。「AGI CPU」という名称は、エージェンティック AI ワークロードを大規模に処理することを意図したもので、汎用人工知能（AGI）そのものを指すわけではない。

### ビジネスモデルの転換

従来 Arm はチップの設計図（IP）をライセンスするビジネスモデルで、自社でチップを製造・販売することはなかった。今回の自社チップ投入は、IP ライセンス料に加えてチップの直接販売収益を得るという大きなビジネスモデルの転換を意味する。

これにより Arm はライセンシーである Qualcomm、MediaTek、Apple などの顧客と競合する立場にもなる。ただしデータセンター向け CPU に特化しているため、モバイル分野での直接的な競合は回避している。

### 顧客と市場の反応

初顧客の Meta は、最大1,350億ドルの設備投資で AI データセンターを大規模に建設中であり、AGI CPU の性能とコスト効率が魅力的だったとみられる。Meta のほか、Cerebras、Cloudflare、F5、OpenAI、Positron、Rebellions、SAP、SK Telecom が顧客として名を連ねる。

株式市場では Arm の株価が発表翌日に16%上昇し、投資家の期待の大きさを示した。Raymond James は買い推奨を維持している。

### AI インフラ競争における位置づけ

Nvidia が GPU で AI 学習・推論市場を支配する中、Arm の AGI CPU は CPU レイヤーでの AI 最適化を狙う。特に推論ワークロードやエージェンティック AI の実行基盤として、GPU を補完する役割が期待される。

Intel と AMD の x86 陣営に対するアーキテクチャレベルでの挑戦でもあり、データセンター CPU 市場の勢力図に影響を与える可能性がある。

## 今後の注目点

- Meta データセンターでの実稼働後の性能ベンチマークと実測データ
- Intel / AMD の x86 陣営からの対抗策
- Arm のライセンシー（特に Ampere Computing）との関係変化
- 2031年の150億ドル収益目標に対する進捗と市場シェア推移
- Nvidia GPU との組み合わせによるシステム全体の最適化事例
