# SpaceX による Cursor（Anysphere）600億ドル買収

- **日付**: 2026-06-20
- **カテゴリ**: Tech
- **ソース**: [CNBC](https://www.cnbc.com/2026/06/16/spacex-spcx-cursor-acquisition-ipo.html) / [Quartz](https://qz.com/spacex-buying-cursor-anysphere-60-billion-deal-061626) / [TheStreet](https://www.thestreet.com/investing/stocks/spacex-acquires-anthropic-and-openai-rival-in-60b-deal)

## 概要

SpaceX は、AI コーディングエディタ「Cursor」を手がける Anysphere を約600億ドルの全株式取引で買収すると発表した。取引完了は規制当局の承認を前提に2026年第3四半期を見込む。今年初めに Elon Musk の AI ベンチャー xAI と統合した SpaceX にとって、Cursor は初の主力コンシューマー／エンタープライズ向けソフト製品となり、Anthropic や OpenAI が提供する人気コーディングツールと直接競合する布石となる。

## 詳細

### 取引の構造と経緯

両社は今年4月に提携を発表し、SpaceX に対し「年内に600億ドルで買収する権利」または「協業対価として100億ドルを支払う」選択肢を与えていた。今回 SpaceX は買収オプションを行使した形となる。全株式取引であり、SpaceX 株での対価となる。

### バリュエーションの急騰

Cursor の直近の資金調達は2025年11月にクローズした23億ドルの Series D で、その時点の評価額は293億ドル。2025年初頭の25億ドルから1年足らずで急騰していたが、今回の600億ドルはそこからさらに倍増した水準であり、AI コーディング領域への投資家の旺盛な期待を映す。創業者の純資産は本件で倍増したと報じられている。

### Cursor とは

Cursor は AI を活用したコードエディタで、ソフトウェア開発を加速する。チャットボット型アシスタント、コード自動補完、コーディングタスクを自律的に処理する AI エージェントなどを備える。開発者の生産性ツールとして急速に普及した。

### 戦略的意義

SpaceX は xAI 統合に続く今回の買収で、AI 能力を一気に拡張する狙い。Anthropic（Claude Code 等）と OpenAI（Codex 等）が主導するエージェント型コーディング市場に対し、自社の計算インフラと組み合わせた垂直統合で対抗する構図となる。同じ週には OpenAI が `uv` / `ruff` を開発する Astral を買収しており、AI コーディングツールチェーンを巡る大手の囲い込みが加速している。

## 今後の注目点

- 第3四半期の取引完了に向けた規制当局（独禁・安全保障）の審査の行方。
- xAI のモデル（Grok 系）と Cursor の統合により、Claude / GPT 依存からどこまで脱却できるか。
- OpenAI による Astral 買収と合わせ、開発者ツールチェーンが特定ベンダーに囲い込まれることへのコミュニティの反応。
- 600億ドルという評価が AI コーディング市場全体のバリュエーション基準を押し上げるか。
