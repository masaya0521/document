# キューバ危機2026 — 全島停電・反政府デモと米キューバ外交の転換点

- **日付**: 2026-03-18
- **カテゴリ**: World
- **ソース**: [Al Jazeera](https://www.aljazeera.com/news/2026/3/14/protests-erupt-in-cuba-as-us-restrictions-spark-food-energy-shortages)

## 概要

2026年3月、キューバが深刻な人道危機に直面している。米国による石油封鎖で1月9日以降石油輸入が途絶え、3月16日には全島規模の停電が発生。約1,100万人が暗闇に取り残された。食料・水・電力の不足に対する抗議デモは10日以上連続で続き、2021年7月11日のデモ以来最大規模と分析されている。一方、ディアス=カネル政権は米国との外交交渉を公式に認め、政治犯51人の釈放に合意するなど、歴史的な転換点を迎えている。

## 詳細

### 危機の経緯

| 時期 | 出来事 |
|------|--------|
| 2026年1月9日 | 米国がベネズエラからキューバ向け石油タンカーを押収。以降、石油輸入が途絶 |
| 2月4日 | 東部4州（グアンタナモ、サンティアゴ・デ・クーバ、オルギン、グランマ）で全面停電 |
| 2月6日 | ディアス=カネル大統領「前提条件なしでワシントンとの対話の用意がある」と表明 |
| 2月 | ニカラグアがキューバ国民のビザ免除渡航を廃止（移民ルートの遮断） |
| 3月4日 | アントニオ・ギテラス火力発電所の停止により西部で大規模停電 |
| 3月7日頃 | 抗議デモが各地で発生開始。カセロラソ（鍋叩き抗議）が夜間に広がる |
| 3月13日 | ディアス=カネルが米国との外交交渉を公式に確認。政治犯51人の釈放に合意 |
| 3月14日 | モロン市で共産党事務所が放火される。5名逮捕 |
| 3月16日 | 全島の電力グリッドが崩壊。約1,100万人が停電下に |
| 3月17日 | トランプ大統領「キューバを取る栄誉がある」と発言 |

### 抗議デモの性質

- 2021年7月11日（「11J」）のデモ以来最大規模
- 食料・水・電力の不足が直接の引き金
- 「カセロラソ」（鍋叩き）が夜間の主要な抗議形態
- 一部では暴力的な衝突に発展（共産党事務所の放火）

### 米国の立場

- トランプ政権は政治犯釈放と政治・経済の自由化を制裁解除の条件に要求
- ルビオ国務長官がキューバの新指導者を求める発言
- トランプ大統領の「キューバを取る」発言は、イラン戦争と並行する積極的外交路線の一環

### 人道的影響

- 石油輸入途絶により燃料・電力供給が壊滅的に不足
- 食料流通の混乱（冷蔵・輸送の停止）
- 医療施設の電力不足
- 水道ポンプの停止による断水

## 今後の注目点

- 米キューバ外交交渉の進展と条件交渉の行方
- 全島停電からの電力復旧のタイムライン
- 抗議デモの拡大・激化の可能性と政府の対応
- 国際社会（特にメキシコ、EU、中国、ロシア）の介入の有無
- 政治犯51人の釈放が実際に履行されるか
- ベネズエラからの石油供給再開の可能性
