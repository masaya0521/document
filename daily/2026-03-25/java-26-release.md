# Java 26 リリース — 主要な新機能と開発者への影響

- **日付**: 2026-03-25
- **カテゴリ**: Tech
- **ソース**: [InfoQ](https://www.infoq.com/news/2026/03/java26-released/)

## 概要

Oracle は2026年3月17日に Java 26（JDK 26）を正式リリースした。10件の JEP（JDK Enhancement Proposals）を含み、そのうち5件が正式版（Final）として提供される。新しい安定版言語機能はないものの、HTTP/3 サポート、パフォーマンス改善、セキュリティ強化など開発者にとって実用的な価値の高いアップデートが含まれる。

## 詳細

### 主要な新機能（Final）

#### 1. HTTP/3 サポート（HTTP Client API）
JDK 26 で最も注目される追加機能。既存の HTTP Client API に HTTP/3 サポートが統合され、最小限のコード変更で HTTP/3 サーバーとの通信が可能になる。QUIC プロトコルベースの HTTP/3 により、接続確立の高速化とヘッドオブラインブロッキングの解消が期待される。

#### 2. AOT オブジェクトキャッシュ（JEP 516）
HotSpot JVM の Ahead-of-Time（AOT）キャッシュを拡張し、ZGC を含む任意のガベージコレクタで動作するようにした。キャッシュされた Java オブジェクトを GC 非依存フォーマットで保存し、起動時にシーケンシャルにメモリへロードする。起動時間の短縮に寄与する。

#### 3. G1 GC スループット改善（JEP 522）
G1 ガベージコレクタにおいて、アプリケーションスレッドと GC スレッド間の同期を削減することでスループットを向上。G1 はデフォルト GC であるため、多くのアプリケーションが恩恵を受ける。

#### 4. Applet API の削除
JDK 17（2021年）で非推奨化された Applet API がついに削除された。現代のブラウザはアプレットをサポートしておらず、長年にわたり実質的に使用されていなかった。

### プレビュー・インキュベーター機能

#### 5. 構造化並行性（Structured Concurrency）
マルチスレッドコードをより直感的に記述するための API。同期的な思考モデルでの並行処理の表現を可能にし、コードの可読性と保守性を向上させる。

#### 6. Lazy Constants
定数畳み込み（constant folding）の対象となる遅延定数のサポート。final フィールドに格納することでコンパイル時最適化の恩恵を受けられる。

#### 7. Vector API（第11回インキュベーション）
CPU のベクトル命令を活用した高性能な数値計算 API。対応 CPU 上でスカラー計算を上回るパフォーマンスを実現する。長期にわたりインキュベーション段階にあるが、着実に改善が続いている。

### セキュリティ強化

- **HPKE（Hybrid Public Key Encryption）**: 業界標準のハイブリッド公開鍵暗号の簡素化
- **ポスト量子対応 JAR 署名**: 量子コンピュータ時代に備えたサプライチェーンセキュリティ
- **Unicode 17.0 / CLDR v48**: グローバル標準への対応更新
- **暗号アルゴリズム制御の強化**: レガシーキーストアの管理改善

## 今後の注目点

- **Java 27（2026年9月予定）**: 次のリリースで構造化並行性や Vector API が正式版になるか
- **HTTP/3 の採用加速**: Spring Boot や各種 HTTP クライアントライブラリでの HTTP/3 サポート対応状況
- **ポスト量子暗号**: JAR 署名以外の領域への拡大（TLS、証明書チェーンなど）
- **AOT キャッシュの本番利用**: コンテナ環境やサーバーレスでの起動時間短縮効果の実測データ
