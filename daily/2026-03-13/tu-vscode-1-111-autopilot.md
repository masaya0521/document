# VS Code 1.111 — 週次リリースと Autopilot モード

- **日付**: 2026-03-13
- **カテゴリ**: Dev Tool
- **ソース**: [VS Code Updates](https://code.visualstudio.com/updates/v1_111), [The Register](https://www.theregister.com/2026/03/11/visual_studio_code_moves_to/)

## 概要

Visual Studio Code 1.111 が 2026年3月9日にリリースされた。本リリースは Microsoft が VS Code を **週次安定版リリース** に移行した最初のバージョンであり、AI エージェント機能を中心とした大幅な機能強化が含まれている。特に注目されるのは、AI エージェントがタスク完了まで自律的に動作する **Autopilot モード** のプレビュー版導入である。

## 主な変更点

### 週次リリースへの移行

従来の月次リリースサイクルから週次安定版リリースへと移行。高品質な機能をより迅速に提供する体制が整備された。これにより、機能の開発から安定版への反映までのリードタイムが大幅に短縮される。

### Autopilot モード（プレビュー）

Copilot Chat の新しい権限レベルとして導入。有効にすると以下の動作が行われる:

- **ツール呼び出しの自動承認**: ユーザーの承認なしにツールを実行
- **エラー時の自動再試行**: 失敗したツール呼び出しを自動的にリトライ
- **質問への自動応答**: ツールが投げかける質問に自動で応答し、エージェントが停止しない
- **タスク完了シグナル**: エージェントが `task_complete` ツールを呼び出して完了を通知

潜在的に破壊的な操作が含まれるため、初回有効化時に警告ダイアログが表示される。

### エージェント権限制御

セッション単位で設定可能な 3 段階の権限レベル:

| レベル | 説明 |
|--------|------|
| Default Approvals | 従来通りツール呼び出しごとに承認 |
| Bypass Approvals | ツール呼び出しを自動承認（質問には応答しない） |
| Autopilot | 完全自律動作（ツール承認 + 質問応答 + エラー再試行） |

### その他のエージェント機能

- **エージェントスコープフック（プレビュー）**: 特定のエージェントにのみ前処理・後処理ロジックを適用。他の Chat インタラクションには影響しない
- **デバッグイベントスナップショット**: `#debugEventsSnapshot` 機能でエージェントの動作をトラブルシュートできる

## 今後の注目点

- 週次リリースの安定性と品質がどう維持されるかが注目される
- Autopilot モードはプレビュー段階であり、安定版への昇格時期は未定
- エージェント権限のきめ細かい制御（特定ツールのみ許可等）が今後追加される可能性
- サードパーティ拡張機能のエージェント対応が進むかどうか
