# Rapidusと日本の半導体復権戦略

- **日付**: 2026-03-08
- **カテゴリ**: Japan Tech
- **ソース**: [Rapidus公式](https://www.rapidus.inc/en/news_topics/information/rapidus-secures-267-6-billion-yen-in-funding-from-japan-government-and-private-sector-companies/)

## 概要

日本の次世代半導体企業Rapidusが2026年2月27日、政府・民間32社から総額2,676億円の追加資金調達を完了した。2027年度後半の2nmロジック半導体量産開始を目指し、日本の半導体産業復権における最重要プロジェクトが大きく前進した。TSMCの熊本工場拡張と合わせ、日本は先端半導体製造の世界的拠点としての地位を再構築しつつある。

## 詳細

### 資金調達の内訳

2026年2月27日に完了した2,676億円の調達は以下で構成される：

- **政府出資（IPA経由）**: 1,000億円
- **民間32社からの出資**: 1,676億円
  - トヨタ自動車、ソニーグループ、キヤノン、ソフトバンク、NTT、富士通、日本政策投資銀行（DBJ）など

この調達完了により、Rapidusの資本金等は累計で2,749億5,000万円に達した。

### 技術的アプローチ

Rapidusが追求する2nm世代は、半導体アーキテクチャの根本的な転換を伴う：

- **GAA（Gate-All-Around）構造**: 現行主流のFinFETから、トランジスタのチャネルを四方からゲートで囲い込む構造へ移行
- **電力効率**: 電流リークの劇的な削減により、消費電力あたりの性能が飛躍的に向上
- **先端パッケージング**: 2026年4月から先端パッケージングの試作を開始予定

2025年7月には、北海道千歳市のIIM-1工場で2nm世代GAAトランジスタの試作に成功している。

### ロードマップ

| 時期 | マイルストーン |
|------|-------------|
| 2025年7月 | 2nm GAA トランジスタ試作成功 |
| 2026年4月 | 先端パッケージング試作開始 |
| 2027年度後半 | 2nm ロジック半導体量産開始 |
| 2028年3月 | 2nm量産の本格化目標 |
| 2029年頃 | 1.4nm プロセスへの移行 |
| 2029年度 | 営業キャッシュフロー黒字化目標 |
| 2031年度 | フリーキャッシュフロー黒字化・上場目標 |
| その後 | 1.0nm プロセスへの移行 |

### 日本全体の半導体戦略

Rapidusの取り組みは、日本政府の半導体復権戦略の中核に位置づけられる：

- **2026年度予算**: AI・半導体分野に約1.23兆円を計上（前年比約300%増）
  - Rapidusへの補助金: 1,500億円
  - 国内AI開発: 3,873億円
- **TSMC熊本工場**: 第2工場で3nmプロセスの製造を計画、2027年後半の量産開始目標
- **2030年までの総投資額**: 政府補助金は約10兆円（650億ドル）規模に達する見通し

### 顧客基盤の構築

Rapidusは60社以上の企業と2nmチップの設計について協議中。主な需要分野は：

- AI（人工知能）向けプロセッサ
- ロボティクス
- エッジコンピューティング

NVIDIAやSoftBank Groupも、地政学リスクを踏まえた台湾・韓国の代替製造拠点として日本に注目していると報じられている。

### 総投資額と課題

2nmプロセスの量産からさらに1.4nmノードの実現までに必要な総投資額は、研究開発費と設備投資を合わせて**7兆円規模**と見積もられている。これは日本の半導体産業への投資としては前例のない規模であり、継続的な資金調達が最大の課題となる。

## 今後の注目点

- **2026年4月の先端パッケージング試作開始**: 量産に向けた技術的マイルストーンの達成状況
- **量産歩留まりの確保**: 2nm世代のGAA構造での量産は世界的にも未達成であり、歩留まり向上が最大の技術的課題
- **追加資金調達の動向**: 7兆円の総投資額に対し、今後も大規模な資金調達が必要
- **TSMCとの棲み分け**: 熊本のTSMC工場（3nm）と千歳のRapidus工場（2nm）の役割分担と日本国内でのサプライチェーン形成
- **地政学リスクへの対応**: 台湾有事リスクを背景とした、日本への半導体製造移転の加速度合い
