# ホルムズ海峡危機 — 世界エネルギー秩序への衝撃

- **日付**: 2026-03-20
- **カテゴリ**: World
- **ソース**: [Wikipedia](https://en.wikipedia.org/wiki/2026_Strait_of_Hormuz_crisis)

## 概要

2026年2月28日、米国・イスラエルによるイランへの軍事攻撃（最高指導者ハメネイ師の殺害を含む）を契機に、イランがホルムズ海峡の封鎖を宣言。世界の海上石油輸送の約20%（日量約2,000万バレル）が通過するこの海峡の閉鎖は、1970年代のオイルショック以来最大のエネルギー供給途絶として世界経済に深刻な影響を及ぼしている。

## 詳細

### 危機の経緯

| 日付 | 出来事 |
|---|---|
| 2月28日 | 米国・イスラエルがイランに軍事攻撃開始。ハメネイ師殺害 |
| 3月2日 | IRGC（イスラム革命防衛隊）がホルムズ海峡の閉鎖を公式発表 |
| 3月5日 | イランが方針転換 — 米国・イスラエルと西側同盟国の船舶のみ通航禁止に |
| 3月8日 | ブレント原油が100ドル/バレルを突破（4年ぶり） |
| 3月12日 | イランによる商船攻撃が21件確認 |
| 3月15日 | EU がアスピデス海軍ミッションのホルムズ海峡への拡大を検討 |
| 3月16日 | イランのドローンがドバイ国際空港を攻撃 |
| 3月18日 | 通航船舶数が倍増の兆し（海事情報企業Windwardのデータ） |
| 3月19日 | 英仏独伊蘭日加の7カ国首脳がホルムズ海峡に関する共同声明を発表 |
| 3月19日 | イスラエルが南パルスガス田（世界最大）を攻撃 |

### エネルギー市場への影響

- **ブレント原油**: 100ドル/バレル突破 → ピーク時126ドル/バレル（2022年7月以来の高値）
- **WTI原油**: 96.14ドル/バレル前後で推移
- **LNG市場**: カタール・UAEからの供給が大幅に減少
- **ブレント原油の直近**: 108.65ドル/バレル（3月19日終値）

### 地域別の脆弱性

最も深刻な影響を受けるのは**南アジア**:

| 国 | カタール・UAE産LNG依存度 |
|---|---|
| パキスタン | 99% |
| バングラデシュ | 72% |
| インド | 53% |

アジア全体が燃料・貨物コストの上昇に直面し、リモートワーカーを含む一般市民の生活にも影響が波及。

### 世界経済への波及

- **インフレ加速**: 燃料・輸送コスト上昇が物価全体を押し上げ
- **金融環境の引き締まり**: FRBは金利据え置き（3.5〜3.75%）を決定するも、インフレ期待の上昇が追加的な金融引き締めリスクに
- **株式市場**: ダウは月間5%超の下落、2022年以来最悪
- **金価格の逆行**: 通常の安全資産としての機能が崩れ、週間10%の急落（1983年以来最悪）
- **農業**: 肥料貿易の途絶が食糧安全保障にも影響
- **景気後退リスク**: 長期化すれば「世界のエネルギー供給の20%なしに世界経済は成長できない」とアナリストが警告

### 国際社会の対応

- **7カ国共同声明**（3月19日）: 英国、フランス、ドイツ、イタリア、オランダ、日本、カナダが航行の自由を要求
- **国連海運機関**: 湾岸で足止めの船員約2万人を救済するための安全な海上回廊の設立を提案
- **EU**: アスピデス海軍ミッション（紅海で展開中）のホルムズ海峡への拡大を検討
- **ウクライナ**: 200名超の軍事専門家が中東でドローン防衛を支援
- **戦略石油備蓄の放出**: 市場安定化の効果は限定的との分析

## 今後の注目点

- イランの選択的通航許可の範囲拡大と、実効的な海峡再開の見通し
- 原油価格の今後の推移と、100ドル超の長期化がもたらすインフレ・景気後退リスク
- 米国・イスラエルの軍事作戦の出口戦略
- 南アジア諸国のLNG供給代替ルートの確保
- 中東のハブ空港（ドバイ等）の運航正常化の時期
- 7カ国共同声明の実効性と、さらなる国際的な連携の動き
- 戦略石油備蓄の放出量とタイミング
