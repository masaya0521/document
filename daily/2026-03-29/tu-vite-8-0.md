# Vite 8.0 — Rolldown 統合による統一バンドラーアーキテクチャ

- **日付**: 2026-03（正式リリース）
- **カテゴリ**: Build Tool
- **ソース**: [公式ブログ](https://vite.dev/blog/announcing-vite8), [The Register](https://www.theregister.com/2026/03/16/vite_8_rolldown/)

## 概要

Vite 8.0 は Vite 2 以来最大のアーキテクチャ変更となるリリースである。開発時は esbuild、本番ビルドは Rollup という二重バンドラー構成を廃止し、Rust 製の統一バンドラー Rolldown に一本化した。これにより開発/本番間の挙動差異が解消され、ビルド速度は 10-30 倍向上している。

## 主な変更点

### Rolldown 統合

- **統一バンドラー**: esbuild + Rollup の二重構成を解消し、Rolldown に一本化
- **プラグイン互換性**: Rollup / Vite の既存プラグイン API を維持。ほとんどの既存プラグインがそのまま動作
- **新機能**: フルバンドルモード、柔軟なチャンク分割、モジュールレベルの永続キャッシュ、Module Federation サポート

### パフォーマンス改善（実測値）

| プロジェクト | 改善幅 |
|-------------|-------|
| Linear | 46秒 → 6秒 |
| Ramp | 57% 削減 |
| Beehiiv | 64% 削減 |

### 新機能

- **統合開発者ツール**: ビルトインの開発者ツール
- **TypeScript パスエイリアス解決**: tsconfig.json のパスエイリアスをネイティブサポート
- **デコレータメタデータ対応**
- **WebAssembly の SSR サポート**
- **ブラウザコンソール転送**: ブラウザのコンソール出力をターミナルに転送

### @vitejs/plugin-react v6

- Babel から Oxc（Rust 製コンパイラ）へ移行
- JSX / デコレータの変換が高速化

### 統合ツールチェーン

Vite 8 により、Vite はエンドツーエンドのツールチェーンのエントリーポイントとなった:
- **ビルドツール**: Vite
- **バンドラー**: Rolldown
- **コンパイラ**: Oxc

パース・解決・変換・ミニファイまで一貫した挙動を保証する。

## 破壊的変更・移行ガイド

### 最低要件

- **Node.js 20.19+ または 22.12+** が必須

### 移行

- ほとんどのプロジェクトでは自動互換性レイヤーが機能し、`vite` パッケージのアップグレードだけで移行可能
- 複雑なプロジェクトには段階的移行を推奨
- インストールサイズは約 15MB 増加（lightningcss と Rolldown のバイナリ分）

### プラグイン互換性

- Rollup プラグイン API と互換のため、大半のプラグインはそのまま動作
- Vite 8 リリースに合わせて Tailwind CSS も `@tailwindcss/vite` で Vite 8 サポートを追加済み

## 今後の注目点

- Rolldown のさらなる最適化と永続キャッシュの改善
- Oxc との統合深化（リンター・フォーマッター等）
- Module Federation の本格活用事例の増加
- VoidZero を中心としたエコシステムの発展
