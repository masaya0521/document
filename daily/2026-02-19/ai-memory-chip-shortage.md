# AIメモリチップ不足危機

- **日付**: 2026-02-19
- **カテゴリ**: Tech
- **ソース**: [Bloomberg](https://www.bloomberg.com/news/articles/2026-02-15/rampant-ai-demand-for-memory-is-fueling-a-growing-chip-crisis), [IEEE Spectrum](https://spectrum.ieee.org/dram-shortage), [Tom's Hardware](https://www.tomshardware.com/pc-components/ram/data-centers-will-consume-70-percent-of-memory-chips-made-in-2026-supply-shortfall-will-cause-the-chip-shortage-to-spread-to-other-segments)

## 概要

AIデータセンターの爆発的な需要増加により、DRAM（動的ランダムアクセスメモリ）が世界的な供給不足に陥っている。2026年に生産されるメモリチップの70%がデータセンターに消費される見通しで、価格は今四半期だけで80-90%上昇。消費者向けデバイスから自動車まで、幅広い産業に波及する深刻な危機となっている。

## 詳細

### 需要側の構造変化

Microsoft、Google、Meta、Amazonなどのハイパースケーラーが、AIモデルの訓練と推論に必要なHBM（High Bandwidth Memory）を大量に確保している。これにより、Samsung Electronics、SK Hynix、Micron Technologyの3大メモリメーカーは、限られたクリーンルーム空間と設備投資を、利益率の高いエンタープライズ向け製品に集中せざるを得ない状況に追い込まれている。

### 価格への影響

- DRAM価格は今四半期で80-90%上昇
- さらに次四半期には40%の追加上昇が予想
- 2026年中に価格が下がる見込みはなし

### 産業への波及効果

AIデータセンターがメモリの大半を消費することで、他のセグメントへの供給が圧迫されている。

- **消費者向けデバイス**: ラップトップ、テレビ等の価格上昇が不可避
- **自動車産業**: Tesla、Apple等の大手企業がDRAM不足による生産制約を示唆
- **データセンター**: 新規構築の遅延リスク

### 供給側の対応

IDCの予測では、2026年のDRAMおよびNAND供給成長率は、歴史的な標準値を下回る前年比16%および17%にとどまる。新たな製造施設（ファブ）の建設が進められているが、本格的な供給改善は2027-2028年まで見込めない状況。

### 2020年代チップ不足との比較

2020-2021年のCOVID-19に起因するチップ不足はロジック半導体が中心だったが、今回の危機はメモリ半導体に集中している。また、前回は需要の急減からの急回復が原因だったのに対し、今回はAIインフラへの構造的な需要シフトが背景にあり、より長期的な問題となる可能性が高い。

## 今後の注目点

- Samsung、SK Hynix、Micronの設備投資計画と新ファブの稼働時期
- HBM4以降の次世代メモリ技術の進展
- 消費者向けデバイス（PC、スマートフォン）の価格転嫁の程度
- 各国政府のメモリ半導体産業支援策（CHIPS法の拡大等）
- AIモデルのメモリ効率改善技術（量子化、スパース化等）が需要を緩和するか
- 2027-2028年の新ファブ稼働による供給改善の実現可能性
