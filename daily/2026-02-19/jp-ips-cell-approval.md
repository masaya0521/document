# iPS細胞再生医療製品、世界初の承認審議へ

- **日付**: 2026-02-19
- **カテゴリ**: Japan Tech
- **ソース**: [NOVAIST](https://novaist.jp/articles/world-first-ips-approval/)

## 概要

厚生労働省は2026年2月19日、iPS細胞由来の再生医療製品2品目について専門家部会で承認の可否を審議する。対象はクオリプス（大阪大学発ベンチャー）の重症心不全向け心筋シート「リハート」と、住友ファーマのパーキンソン病向け「アムシェプリ」。部会で了承され厚労相が最終承認すれば、iPS細胞を用いた再生医療製品として世界初の承認事例となる。2006年の山中伸弥教授によるiPS細胞作製報告から20年の節目を迎える年に、日本発の技術が実用化の最終段階に到達した。

## 詳細

### 審議対象の2製品

#### リハート（クオリプス）

- **適応疾患**: 重症心不全
- **製品概要**: iPS細胞から分化させた心筋細胞をシート状に加工し、弱った心臓の表面に貼り付ける。移植された心筋シートが拍動を補助し、血流改善を期待する治療法
- **開発元**: クオリプス（大阪大学 澤芳樹教授の研究をもとに設立されたベンチャー企業）
- **臨床経緯**: 大阪大学での臨床研究を経て製造販売承認申請に至った

#### アムシェプリ（住友ファーマ）

- **適応疾患**: パーキンソン病
- **製品概要**: iPS細胞からドパミン産生神経前駆細胞を分化させ、脳内に移植する。パーキンソン病で失われたドパミン関連神経細胞を補充し、運動症状の改善を目指す
- **開発元**: 住友ファーマ（京都大学 高橋淳教授との共同研究がベース）
- **臨床経緯**: 医師主導治験を経て承認申請

### 審査の焦点

iPS細胞由来製品の審査では、従来の医薬品とは異なる固有のリスク評価が求められる。

- **腫瘍化リスク**: iPS細胞は多能性を持つため、未分化細胞が残存した場合の腫瘍形成リスクの評価が重要
- **移植後の細胞動態**: 移植された細胞が体内でどのように定着・機能するかの長期的な挙動の評価
- **製造品質**: 細胞製品特有の品質管理基準。ロット間のばらつき管理や、医薬品並みの製造基準（GMP）への適合
- **免疫拒絶**: 他家（他人の）iPS細胞を使用する場合の免疫抑制の必要性と安全性

### iPS細胞研究の20年

| 年 | 出来事 |
|-----|--------|
| 2006年 | 山中伸弥教授がマウスiPS細胞の作製に成功 |
| 2007年 | ヒトiPS細胞の作製に成功 |
| 2012年 | 山中教授がノーベル生理学・医学賞を受賞 |
| 2014年 | 理研が世界初のiPS細胞由来網膜細胞の移植手術を実施 |
| 2018年 | 京大がiPS細胞由来ドパミン神経前駆細胞のパーキンソン病患者への移植を開始 |
| 2020年 | 大阪大学がiPS心筋シートの臨床研究を開始 |
| 2026年 | 世界初のiPS細胞再生医療製品の承認審議へ |

### 世界的な意義

承認されれば、日本が「iPS細胞の発見」から「臨床実用化」まで一貫してリードしてきた分野で、世界に先駆けて製品化を達成することになる。これは日本の再生医療産業の国際競争力を示すマイルストーンであるとともに、iPS細胞技術の実用性を世界に証明する意味を持つ。

### 産業への波及効果

再生医療製品の承認は、関連する製造技術、品質管理、サプライチェーンの整備を加速させる。iPS細胞の製造・培養・分化技術を持つ企業群にとって、初の商用製品の承認はビジネスモデルの実証となり、投資を呼び込む契機となりうる。

## 今後の注目点
