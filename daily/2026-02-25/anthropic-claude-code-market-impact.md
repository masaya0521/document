# Anthropic Claude Codeが引き起こした市場激震 — IBM・サイバーセキュリティ株への波及

- **日付**: 2026-02-25
- **カテゴリ**: Tech
- **ソース**: [CNBC](https://www.cnbc.com/2026/02/23/ibm-is-the-latest-ai-casualty-shares-are-tanking-on-anthropic-cobol-threat.html), [Bloomberg](https://www.bloomberg.com/news/articles/2026-02-20/cyber-stocks-slide-as-anthropic-unveils-claude-code-security)

## 概要

2026年2月、Anthropicが発表したClaude Codeの新機能群が複数の市場セクターに激震を走らせた。2/20に発表された「Claude Code Security」はサイバーセキュリティ株を急落させ、2/23にはClaude CodeのCOBOL近代化能力がIBM株を2000年以来最大の暴落に追い込んだ。AIツールが既存の大企業のビジネスモデルを直接脅かす事例として、市場に大きな衝撃を与えている。

## 詳細

### Claude Code Security — サイバーセキュリティ業界への衝撃

2/20、Anthropicは「Claude Code Security」を発表した。これはClaude Opus 4.6を活用した自律型脆弱性検出システムで、コードベースのセキュリティ脆弱性を自動スキャンし、パッチを提案する。

**主な特徴:**
- 内部ストレステストで、本番稼働中のオープンソースコードベースから500件以上の脆弱性を特定
- 従来の手動レビューでは見逃されがちな微妙なロジック欠陥を検出
- リアクティブな監視からプロアクティブな自動修復へのパラダイムシフトを提示

**サイバーセキュリティ株への影響（2/20）:**

| 銘柄 | ティッカー | 下落幅 |
|------|-----------|--------|
| Okta | OKTA | -9.2% |
| SailPoint | SAIL | -9.1% |
| CrowdStrike | CRWD | -6.8% |
| Cloudflare | NET | -6.7% |
| Zscaler | ZS | -3.5% |
| Global X Cybersecurity ETF | BUG | -4.9% |

### COBOL近代化 — IBMビジネスモデルへの直接的脅威

2/23、AnthropicはClaude CodeがCOBOL近代化を大幅に自動化できることを示した。

**技術的なインパクト:**
- 数千行のコードにまたがる依存関係のマッピング、ワークフローの文書化、リスクの特定を自動化
- 従来、人間のアナリストが数ヶ月かけていた作業をAIが短時間で実行
- Anthropicの主張: 「レガシーコードの近代化は、理解コストが書き直しコストを上回っていたために停滞していた。AIがその方程式を反転させる」

**IBM株への影響:**
- **終値**: $223.35（-13.2%）— 2000年以来最大の1日下落
- **時価総額消失**: $310億以上
- **年初来下落**: 24%超
- COBOL近代化はIBMのコンサルティング・サービス事業の収益柱の一つ

### 市場の文脈 — 「AI不安」の連鎖

これらの動きは孤立した事象ではなく、AI技術がSaaS・サービス業のビジネスモデルを根本から変える可能性に対する市場全体の不安（「AI Anxiety」）の一部として解釈されている。

- 投資家は「まず売って、後から考える」(sell first, ask questions later) モードに
- サブスクリプション型ビジネスモデルへのAI代替の脅威が広く意識される
- Jim Cramerは「AI不安は過剰反応」と指摘する一方、Jefferiesのアナリストは「AIの実際の影響が明確になるまで、ヘッドライン駆動の下落は続く」と警告

## 今後の注目点

- **Claude Code Securityの実際の導入状況**: エンタープライズでの採用が進むか、それとも従来のセキュリティツールとの併用に留まるか
- **IBMの対応戦略**: COBOL近代化事業をAIと共存させる戦略の発表はあるか
- **競合他社の動向**: Google、Microsoft、OpenAIがAIセキュリティ・コード近代化分野で対抗策を打ち出すか
- **2026年Q1決算シーズン**: サイバーセキュリティ企業の業績にAIツールの影響が実際に現れるか
- **規制・ライセンスの問題**: AIが自動生成したセキュリティパッチの法的責任やコンプライアンス上の課題
