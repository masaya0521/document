# 衆院選圧勝後の第2次高市内閣と日本政治の行方

- **日付**: 2026-02-23
- **カテゴリ**: Japan General
- **ソース**: [日本経済新聞](https://www.nikkei.com/article/DGXZQOUA0861M0Y6A200C2000000/)

## 概要

2026年2月8日に投開票された第51回衆議院議員総選挙で、自民党が戦後最多の316議席を獲得して歴史的な圧勝を収めた。2月18日に召集された特別国会で高市早苗首相が第105代首相に選出され、第2次高市内閣が発足。全閣僚を再任し、積極財政と安全保障強化路線の継続を明確にした。

## 詳細

### 衆院選の結果

2026年1月27日公示、2月8日投開票の衆院選（定数465）で、自民党は単独で3分の2を超える316議席を獲得した。連立パートナーの日本維新の会（36議席）と合わせて与党352議席という巨大与党が誕生した。

#### 各党の獲得議席

| 政党 | 議席数 | 備考 |
|------|--------|------|
| 自民党 | 316 | 単独で3分の2超、戦後最多 |
| 日本維新の会 | 36 | 連立与党 |
| 中道改革連合 | 49 | 公示前167から大幅減 |
| 国民民主党 | 28 | - |
| 参政党 | 15 | 躍進 |
| チームみらい | 11 | 初の議席獲得（比例のみ） |
| 共産党 | 4 | - |
| れいわ新選組 | 1 | - |
| その他 | 5 | - |

自民党の従来の最多議席は1986年の304議席であり、今回は単独政党の獲得議席として過去最多を更新した。

### 野党の惨敗

最大野党の中道改革連合は公示前の167議席から49議席に激減し、壊滅的な敗北を喫した。小選挙区ではわずか7議席にとどまり、比例代表の42議席で何とか存在感を保った形となった。野田代表は敗北を受けて辞任を表明し、代表選の実施が決定した。

### 新勢力の台頭

AIエンジニア主導の新党「チームみらい」（安野貴博代表）が比例代表で11議席を獲得し初の国政進出を果たした。AI活用による低賃金外国人労働者依存の削減や高度人材受け入れ強化を掲げ、テクノロジーと政治の融合を体現する存在として注目を集めた。参政党も15議席に躍進し、既存野党とは異なる新しい政治勢力の存在感が増した。

### 第2次高市内閣の発足

2月18日の特別国会で高市首相が第105代首相に選出され、第2次高市内閣が発足した。高市政権は2025年10月の発足から約4カ月と短期間であったため、全閣僚を再任する安定重視の人事となった。

### 主要政策の方向性

- **積極財政**: 経済成長を最優先とし、大規模な財政出動を継続
- **安全保障強化**: 防衛費の拡充と日米同盟の深化
- **外国人労働者**: 1月23日の閣議決定で2029年3月までに19分野で約123万人の外国人労働者受け入れ方針を示す
- **予算成立**: 年度内（3月末まで）の来年度予算成立を目指す姿勢を表明

### 選挙の争点

- 物価高対応を含む経済政策
- 外国人労働者の受け入れ政策
- 消費税の減税・廃止の是非
- ガソリン暫定税率廃止の影響

- チームみらい・参政党など新勢力の国会での活動
- 参議院選挙（2028年予定）に向けた各党の戦略
