# Android Show I/O Edition 2026 — Android 17 と XR への布石

- **日付**: 2026-05-12
- **カテゴリ**: Tech
- **ソース**: [9to5Google](https://9to5google.com/2026/05/05/android-show-i-o-2026/), [Android.com — New features in 2026](https://www.android.com/new-features-on-android/io-2026/), [Android Authority](https://www.androidauthority.com/the-android-show-2026-google-io-3663492/)

## 概要

Google は 2026 年も I/O 本編（5/19-20、マウンテンビュー）の 1 週間前に「The Android Show: I/O Edition」をオンライン開催する。本日 5/12 13:00 ET（22:00 JST）から YouTube でライブ配信され、Android 17、Gemini との深い統合、Android XR、新プラットフォーム「Aluminium OS」など、Android プラットフォームの「これまでで最大の年」の幕開けを宣言する。ライブキーノートというより事前収録のショーケース形式で、I/O 本編で発表する詳細の地ならしという位置づけ。

## 詳細

### 期待される発表内容

#### 1. Android 17

リーク・ベータビルドから推測される主要変更点：

- **マルチタスク強化**: 大画面端末（タブレット、折りたたみ）でのアプリ並列実行、ドラッグ&ドロップ周りの API 改善。
- **画面録画ツール**: システムレベルでの録画品質設定、ハイライト自動抽出機能。
- **大画面向け UI 改善**: フォルダブル・タブレットでの SystemUI 一新、Material 3 Expressive のさらなる適用。
- **プライバシー**: Permission Hub 拡張、AI モデル経由のデータアクセス可視化。

#### 2. Gemini 統合

Android Show 2025 で示された Gemini Nano のオンデバイス展開がさらに進む。
- **Live API**: アプリから音声・映像のマルチモーダル入力で Gemini を呼ぶ標準 API。
- **Agentic Actions**: Gemini が他アプリの UI を操作してタスクを完遂する仕組み（昨年の Project Astra の延長）。
- **オンデバイス推論**: Pixel 11 シリーズの NPU を活かした完全ローカル推論モード。

#### 3. Android XR

Samsung と協業する Project Moohan ヘッドセットの開発者向け SDK 公開が予想される。
- **XR Glasses**: 軽量グラス型デバイスの仕様、複数 OEM（XReal、Samsung 等）展開。
- **Mixed Reality 開発**: Jetpack XR、Unity 連携、ARCore からの移行パス。
- **空間 UI**: Compose for XR の安定版リリース。

#### 4. Aluminium OS

リークで噂される新しい Android 派生 OS。詳細は不明だが、ChromeOS との統合、または ChromeOS 後継としての可能性が指摘されている。Google が「Android、ChromeOS、Wear OS、Auto、TV」のすべてを統一プラットフォーム上に再構築する戦略の中核と推測される。

### 「I/O 前哨戦」フォーマットが定着した意義

2025 年に始まった Android Show は、本来 I/O キーノートで 3 時間にわたって流していた Android 関連発表を切り出すことで、

1. **I/O 本編を AI／インフラ／開発者ツール中心にフォーカスできる**
2. **Android 単独で記者・開発者の関心を引きつけ、メディア露出時間を最大化できる**
3. **Pixel・XR・Wear などプロダクト発表を I/O より柔軟な日程で実施できる**

という効果を生んでいる。本年も「one of the biggest years for Android yet」と Google が前置きする以上、相応の物量で I/O 本編との二段ロケットを狙うとみられる。

### iOS との競争軸

直近の Apple は WWDC（6 月）に向けて Apple Intelligence の機能拡充に注力。Gemini が iPhone に統合される可能性も WWDC 直前まで噂が絶えない。Google にとって本日のショーは、

- **「Android = AI ファースト」のメッセージ**
- **オンデバイス AI で iPhone を上回る速度・低レイテンシの提示**
- **XR でゴーグル領域での主導権確保**

を狙う場となる。

## 今後の注目点

- **5/19-20 Google I/O 本編**: Android Show で予告された機能の詳細実装、開発者 SDK、Gemini API の新機能。
- **Android 17 Developer Preview / Beta スケジュール**: 通常 6 月に最初の Beta、秋（9-10 月）に安定版リリース。
- **Pixel 11 / 11 Pro 発表時期**: 8 月の Made by Google イベントが本命。
- **Android XR デバイス価格と発売日**: Samsung Moohan の量産時期、Vision Pro 比でのプライシング。
- **Aluminium OS の正体**: ChromeOS 統合の具体像、サードパーティ OEM の対応状況。
- **EU DMA 規制への対応**: Android が引き続きゲートキーパー指定下にある中で、Gemini デフォルトアシスタント化との両立。
