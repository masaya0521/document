# Anthropic Claude CodeのCOBOL参入とソフトウェア業界への衝撃

- **日付**: 2026-02-24
- **カテゴリ**: Tech
- **ソース**: [CNBC](https://www.cnbc.com/2026/02/23/ibm-is-the-latest-ai-casualty-shares-are-tanking-on-anthropic-cobol-threat.html) / [Fortune](https://fortune.com/2026/02/06/anthropic-claude-opus-4-6-stock-selloff-new-upgrade/)

## 概要

Anthropicが2月23日にClaude CodeのCOBOLモダナイゼーション機能を発表し、IBM株が13%急落した。これは2000年以来最大の一日下落率であり、2月単月では24%超の下落となった。この動きは、2月初旬のClaude Coworkプラグインによる「SaaSpocalypse（SaaS大崩壊）」に続く、AI企業による既存ソフトウェア業界への破壊的影響の最新事例である。

## 詳細

### Claude CodeのCOBOL機能

Anthropicが発表したClaude CodeのCOBOL関連機能は以下の通り：

- **依存関係マッピング**: 数千行のCOBOLコードにわたる依存関係を自動マッピング
- **ワークフロー文書化**: 業務フローの自動解析とドキュメント化
- **リスク特定**: 人間のアナリストが要する時間を大幅に短縮してリスクを特定
- **モダナイゼーション支援**: 探索・分析フェーズの自動化により、最も複雑な部分の工数を削減

### なぜIBMが標的になったのか

COBOLは1959年に開発された言語だが、現在も金融業界の基幹システムで広く使用されている：

- **米国のATM取引の約95%** がCOBOLベースのシステムで処理
- IBMはメインフレームシステムとCOBOL関連サービスで長年市場を支配
- レガシーシステムの保守・モダナイゼーションはIBMの重要な収益源

AIがこの領域を自動化できるとなれば、IBMのビジネスモデルの根幹が脅かされる。

### 2月のソフトウェア株売却の時系列

| 日付 | イベント | 影響 |
|------|---------|------|
| 2月初旬 | Claude Cowork業界別プラグイン発表 | SaaS株2,850億ドル相当の売却（「SaaSpocalypse」） |
| 2月6日 | Claude Opus 4.6発表 | FactSet(-10%)、S&P Global、Moody's、Nasdaq急落 |
| 2月21日 | Claude Code Security機能発表 | サイバーセキュリティ株下落（CrowdStrike -10%） |
| 2月23日 | Claude Code COBOL機能発表 | IBM -13%、ソフトウェアETF 52週安値 |

### 業界全体への波及

- **iShares拡張テクノロジーソフトウェアETF**: 52週安値（-5%超）
- **Microsoft**: -3%
- **CrowdStrike**: -10%近い下落
- **Salesforce、Adobe、DocuSign**: いずれも52週安値を更新

### AI破壊への懸念

Citrini Researchは、AIブームが10%の失業率につながる可能性があるとするレポートを発表。AIが知識労働を自動化する速度と範囲が、従来のテクノロジーサイクルよりもはるかに広範であるとの懸念が広がっている。

一方、Gartnerのアナリストは「SaaS企業の死亡予測は時期尚早」と反論し、既存の業務プロセスの転換には時間がかかると指摘している。

## 今後の注目点

- **NVIDIA決算（2月26日）**: AI投資の実態を示す最重要指標。好決算ならAIインフラ需要の持続を裏付け
- **Anthropicの次の動き**: Claude Codeの対象領域がさらに拡大するかどうか
- **IBMの対抗戦略**: watsonx等の自社AI戦略で反撃できるか
- **SaaS企業のAI統合**: 既存SaaS企業がAI機能を統合して差別化できるかが生存の鍵
- **雇用市場への影響**: COBOL技術者の需要変化と、より広範なソフトウェアエンジニアリング職への影響
- **150日後の関税期限との交差**: ソフトウェア株の回復がマクロ環境に左右される可能性
