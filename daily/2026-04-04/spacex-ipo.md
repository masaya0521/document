# SpaceX IPO — 史上最大の新規上場に向けた動き

- **日付**: 2026-04-04
- **カテゴリ**: Tech
- **ソース**: [CNBC](https://www.cnbc.com/2026/04/01/spacex-confidentially-files-for-ipo-setting-stage-for-record-offering.html), [Bloomberg](https://www.bloomberg.com/news/articles/2026-04-01/spacex-is-said-to-file-confidentially-for-ipo-ahead-of-ai-rivals)

## 概要

SpaceXが2026年4月1日にSECへ機密IPO申請を提出した。2月のxAI統合後の評価額$1.25Tをベースに、$1.75T超の評価額と最大$75Bの調達を目指す。実現すれば米国IPO史上最大の規模となり、6月のNasdaq上場を予定している。

## 詳細

### 事業構成

SpaceXは以下の事業を統合した巨大エンティティとしてIPOに臨む:

| 事業 | 概要 |
|------|------|
| SpaceX（宇宙事業） | 2008年以降$24.4Bの政府契約を獲得した航空宇宙・防衛事業 |
| Starlink | 約10,000基の衛星を運用する衛星インターネットサービス |
| xAI | 2026年2月に統合したAI企業。AI開発とインフラ |
| X（旧Twitter） | ソーシャルネットワーク。xAI統合の一部として組み込み |

### IPOの規模感

| 項目 | 数値 |
|------|------|
| 目標評価額 | $1.75T超 |
| 調達目標額 | 最大$75B |
| 過去最大の米国IPO比 | 3倍以上 |
| 統合前評価額（xAI合併後） | $1.25T |

### スケジュール

- **2026年4月1日**: SEC機密IPO申請
- **2026年4月下旬〜5月**: 公開S-1提出予定
- **2026年6月**: Nasdaqでの上場・取引開始予定

公開S-1はIPOロードショー開始の15日前までに提出が必要。

### 株式構造

デュアルクラス株式構造を検討中。マスク氏やインサイダーに不均衡な議決権を付与する設計で、上場後もマスク氏が経営の実権を維持する狙いがある。

### 市場環境のリスク

中東紛争に起因する地政学的不確実性と市場のボラティリティが、IPOのタイミングに影響を与える可能性がある。一方で、SpaceXへの投資機会の希少性（「今後5年間でこのような企業が5社も上場することはない」）から、強い個人投資家需要が見込まれている。

## 今後の注目点

- **公開S-1の内容**: 各事業セグメントの売上・利益構造が初めて開示される。特にStarlinkの収益性とxAIの事業規模に注目
- **デュアルクラス構造への市場反応**: ガバナンス面での批判がIPO需要に影響するか
- **中東情勢との兼ね合い**: 6月上場予定だが、原油危機の長期化で市場環境が悪化した場合の延期リスク
- **AI事業の評価**: xAI統合がSpaceXの評価額にどの程度プレミアムを加えているか、投資家の評価が分かれるポイント
- **競合IPOとの関係**: OpenAI等のAI企業IPOとの資金争奪戦の行方
