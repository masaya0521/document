# OpenAI が80年未解決のErdős予想を自律的に反証

- **日付**: 2026-05-26
- **カテゴリ**: Tech
- **ソース**: [OpenAI](https://openai.com/index/model-disproves-discrete-geometry-conjecture/)

## 概要

2026年5月20日、OpenAI は内部の推論モデルが1946年にPaul Erdősが提出した平面単位距離問題（square grid conjecture）を自律的に反証したと発表した。AIが著名な未解決数学問題を人間の介入なしに独立して証明した初の事例であり、Fields賞受賞者の Tim Gowers が「AI数学におけるマイルストーン」と評した。

## 詳細

### 問題の内容

平面単位距離問題は「平面上にn個の点を配置したとき、ちょうど距離1離れたペアの最大数はいくつか」という問いである。80年間、正方格子（square grid）が本質的に最適であると信じられてきた。

### AIによる解法

OpenAIの推論モデルは、深い代数的整数論のツール、具体的には無限類体塔（infinite class field towers）と1960年代の Golod-Shafarevich の定理を用いて、正方格子を上回る構成を発見した。

重要なのは以下の点である:
- このモデルはこの問題のために特別に訓練されたものではない
- 既存の解法を検索・参照していない
- 人間がステップバイステップで誘導していない
- 問題文を受け取り、独立して証明を生成した

### 検証

Princeton の Noga Alon、Melanie Wood、Thomas Bloom の3名の数学者が独立して結果を検証し、証明の正当性を確認した。

### 意義

これまでのAI数学の成果は、主にコンペティション問題や特定タスクに最適化されたシステムによるものだった。今回は汎用推論モデルが、数学分野の中心的な未解決問題を自律的に解いた初の事例であり、質的に異なるブレークスルーとなる。

## 今後の注目点
