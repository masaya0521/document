# Anthropic Project Glasswing: AIによるOSS脆弱性検出の新時代

- **日付**: 2026-05-26
- **カテゴリ**: Tech
- **ソース**: [Anthropic](https://www.anthropic.com/research/glasswing-initial-update)

## 概要

Anthropic が2026年4月7日に立ち上げた Project Glasswing が、最初の1ヶ月で10,000件以上の高・重大脆弱性を発見した。未公開のフロンティアモデル Claude Mythos Preview を用いて、システム的に重要なソフトウェアのセキュリティ強化を図るプロジェクトである。

## 詳細

### プロジェクトの仕組み

Project Glasswing は、重要なオープンソースのコードベースのメンテナーに対し、脆弱性をプロアクティブに特定・修正できるAIモデルへのアクセスを提供する。約50のパートナー組織と連携し、協調的脆弱性開示（Coordinated Vulnerability Disclosure）のプロセスに則って運用される。

### 検出実績（開始1ヶ月）

- **総検出数**: 10,000件以上の高・重大脆弱性
- **OSSプロジェクト**: 1,000プロジェクトから6,202件の高・重大脆弱性（総計23,019件中）
- **開示状況（5月22日時点）**: 281のOSSプロジェクトに対し1,596件を開示、うち97件がパッチ済み、88件にCVE/GHSAが割り当て

### パートナーの成果

- **Cloudflare**: 2,000件のバグ検出（うち400件が高・重大）
- **Mozilla Firefox**: 271件の脆弱性を修正（以前のClaudeモデルによる検出の10倍）
- **wolfSSL**: 証明書偽造を可能にする脆弱性を検出（攻撃者が銀行やメールプロバイダの偽サイトをホストできる）

### Claude Mythos Preview

Project Glasswing の中核を担う未公開のフロンティアモデル。セキュリティ脆弱性の検出に特に高い能力を持ち、従来のClaude モデルと比較して大幅に検出精度が向上している。

## 今後の注目点

- パートナー数の拡大と対象プロジェクトの範囲
- 検出された脆弱性のパッチ率の推移
- Claude Mythos Preview の一般提供の可能性
- 攻撃者がAIを脆弱性発見に使う「攻防の非対称性」への対応
- 他社（Google Project Zero、Microsoft等）との連携や競合
- OSSメンテナーの負担増加（大量の脆弱性報告への対応）への対策
