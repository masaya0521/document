# Microsoft、自社製 AI モデル群 MAI を全面展開 — 脱 OpenAI の本格化

- **日付**: 2026-06-07
- **カテゴリ**: Tech
- **ソース**: [TheAITrack](https://theaitrack.com/ai-news-june-2026-in-depth-and-concise/) / [BuildFastWithAI](https://www.buildfastwithai.com/blogs/ai-news-today-june-6-2026)

## 概要

Microsoft は自社開発の AI モデル群「MAI ファミリー」を発表した。推論フラッグシップの MAI-Thinking-1 を中心に、コーディング、画像生成、音声認識・合成まで幅広い 7 モデルで構成され、いずれも OpenAI ではなく Microsoft が自社で訓練した。OpenAI への戦略的依存を引き下げ、自前の AI スタックを構築する動きが鮮明になった。

## 詳細

### MAI ファミリーの構成

公表されたモデルは以下を含む 7 種。

- **MAI-Thinking-1** — 推論特化のフラッグシップ
- **MAI-Code-1-Flash** — コーディング向けモデル
- **MAI-Image-2.5** — 画像生成
- **MAI-Transcribe-1.5** — 音声書き起こし
- **MAI-Voice-2** — 音声合成
- 加えて推論特化の追加 2 モデル

すべて社内で訓練された点が強調されており、Microsoft が基盤モデルの内製能力を本格的に確立したことを示す。

### 戦略的背景

Microsoft はこれまで OpenAI への巨額出資と独占的なクラウド提供を通じて AI 戦略を構築してきたが、コスト・供給・交渉力の観点から自社モデルへの分散を進めている。MAI ファミリーは Copilot などの自社製品に組み込まれることで、外部 API への依存とコストを抑える狙いがあるとみられる。同時期に Microsoft は Azure Linux 4.0 のパブリックプレビューや Build 2026 での開発者向け施策も打ち出しており、AI とインフラの両面で内製化・統合を加速している。
