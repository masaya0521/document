# 公取委の日本マイクロソフト立入検査とクラウド市場競争

- **日付**: 2026-02-26
- **カテゴリ**: Japan Tech
- **ソース**: [日本経済新聞](https://www.nikkei.com/article/DGXZQOUD125G90S6A210C2000000/)

## 概要

公正取引委員会は2026年2月25日、日本マイクロソフト（東京・港）を独占禁止法違反（不公正な取引方法）の疑いで立入検査を実施した。マイクロソフトのクラウド基盤「Azure」を巡り、顧客が競合クラウドサービスを利用することを不当に妨げた疑いが強まっている。この検査は、世界各国の競争当局がマイクロソフトのクラウドライセンス慣行を問題視する動きの一環であり、日本のクラウド市場の競争環境に大きな影響を及ぼす可能性がある。

## 詳細

### 疑われている行為

公取委が問題視しているのは、マイクロソフトの以下のようなライセンス慣行である：

- **競合クラウドでの利用制限**: Windows ServerやMicrosoft 365などの自社ソフトウェアを、Amazon Web Services（AWS）やGoogle Cloudなど競合のクラウドサービス上で利用する際に、技術的な制限を設けたり利用自体を認めなかったりしている疑い
- **差別的な料金体系**: 自社クラウド「Azure」以外の環境でマイクロソフト製品を利用する場合に、割高なライセンス料金を設定している疑い
- **Azure誘導**: これらの慣行により、顧客が実質的にAzureを選択せざるを得ない状況を作り出し、クラウド市場の公正な競争を阻害している疑い

### グローバルな規制動向

日本の公取委による検査は、世界的なマイクロソフトのクラウドライセンス慣行への規制強化の流れの中で行われている：

- **英国（CMA）**: 2025年7月にクラウドインフラ市場の最終報告を公表。競争が「十分に機能していない」と結論付け、マイクロソフトのライセンス慣行を競争阻害要因の一つとして指摘
- **ブラジル（CADE）**: 2026年1月にマイクロソフトのライセンス・クラウド政策に関する行政調査を開始。CMAの調査結果を根拠として明示的に引用
- **米国・EU**: 複数の競争当局がマイクロソフトに対する調査を並行して進行中

### 日本のクラウド市場の状況

日本のパブリッククラウド市場は急速に成長しており、企業のDX推進に伴いクラウド移行が加速している。市場ではAWS、Azure、Google Cloudの3大プロバイダーが大きなシェアを占めるが、ライセンス条件によるベンダーロックインが問題視されてきた。公取委の介入は、マルチクラウド戦略を推進する企業にとって競争環境の改善につながる可能性がある。

### マイクロソフトの対応

日本マイクロソフトは公取委の要請に「全面的に協力する」とコメントしている。米国本社の対応も含めて、今後の実態解明が進められる見通しである。

## 今後の注目点

- 公取委による押収資料の分析と関係者聞き取りの進捗、排除措置命令が出されるかどうか
- マイクロソフトがライセンス条件を自主的に見直す可能性（EUでは過去に自主的な改善が行われた前例あり）
- 他の大手クラウドプロバイダー（AWS、Google Cloud）のライセンス慣行への波及調査の可能性
- 日本のクラウド市場における競争政策の方向性と、デジタルプラットフォーム規制との関連
- グローバルでの同時並行的な規制強化が、マイクロソフトのクラウド事業戦略に与える影響
