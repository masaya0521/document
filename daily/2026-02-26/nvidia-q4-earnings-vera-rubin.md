# NVIDIA FY2026 Q4決算とVera Rubinチップの展望

- **日付**: 2026-02-26
- **カテゴリ**: Tech
- **ソース**: [CNBC](https://www.cnbc.com/2026/02/25/nvidia-nvda-earnings-report-q4-2026.html)

## 概要

NVIDIAは2026年2月25日にFY2026第4四半期（2025年11月〜2026年1月）決算を発表した。データセンター事業が前年同期比75%増の623億ドルと驚異的な成長を記録し、全社売上高は681.3億ドル（アナリスト予想662.1億ドルを上回る）に達した。また、次世代AIプラットフォーム「Vera Rubin」の初期サンプル出荷が顧客に開始されたことを明らかにし、2026年後半の本格出荷に向けた計画が順調に進行していることを示した。

## 詳細

### 決算ハイライト

| 項目 | 実績 | 前年同期比 |
|------|------|-----------|
| 総売上高 | $681.3億 | 大幅増 |
| 調整後EPS | $1.62 | ほぼ倍増 |
| 純利益 | $430億 | 約95%増 |
| データセンター売上 | $623億 | +75% |
| ネットワーキング売上 | $109.8億 | +263% |
| ゲーミング売上 | $37億 | +47% |
| プロフェッショナル可視化 | $13.2億 | +159% |
| 自動車 | $6.04億 | +6% |

### セグメント分析

データセンター事業が全社売上の91%以上を占める構造がさらに強化された。特にハイパースケーラー（AWS、Azure、Google Cloudなど）が売上の50%以上を占め、大手テック企業のAIインフラ投資が依然として旺盛であることが確認された。

ネットワーキング部門の263%増は、AIクラスターにおける高速インターコネクトの需要急増を反映している。NVLink技術の採用拡大がこの成長を牽引した。

### 来期ガイダンス

FY2027 Q1の売上高ガイダンスは780億ドル（±2%）で、アナリスト予想の726億ドルを大幅に上回った。中国向けデータセンター売上は含まれていない点に留意が必要。

### Vera Rubinプラットフォーム

Vera Rubinは、現行のBlackwellアーキテクチャの後継となるNVIDIAの次世代AIプラットフォームであり、CES 2026で発表された。

**主要スペック:**

- **Rubin GPU**: 3,360億トランジスタ、2つのレチクルダイで構成
  - 推論性能: 50 PFLOPS（NVFP4）— Blackwell比5倍
  - 学習性能: 35 PFLOPS（NVFP4）— Blackwell比3.5倍
  - メモリ: HBM4、GPU当たり最大288GB、帯域幅22 TB/s

- **Vera CPU**: 2,270億トランジスタ、カスタムArm「Olympus」コア
  - 88コア / 176スレッド（Spatial Multi-Threading）
  - メモリ: 最大1.5TB LPDDR5x、帯域幅1.2 TB/s

- **NVL72ラックスケールシステム**: 72 GPU + 36 CPU
  - 推論: 3.6 EFLOPS（NVFP4）
  - 学習: 2.5 EFLOPS
  - HBM4容量: 20.7TB
  - 推論トークンコスト: Blackwell比1/10
  - MoE学習に必要なGPU数: Blackwell比1/4

**出荷スケジュール:**
- 初期サンプル: 2026年2月（顧客に提供開始）
- 本格出荷: 2026年後半

### 電力効率とインフラ課題

Vera Rubinは消費電力がBlackwellの約2倍となるが、性能あたりの電力効率は10倍向上する。これは、データセンターの電力制約が世界的に深刻化する中で極めて重要な指標となる。AIインフラの電力需要は送電網の拡張ペースを大きく上回っており、効率改善はNVIDIAの競争力の中核要素である。

## 今後の注目点

- **OpenAIとの提携**: Jensen Huang CEOが「合意に向けた進捗」に言及。詳細は未発表
