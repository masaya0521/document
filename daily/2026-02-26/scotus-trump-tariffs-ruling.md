# 米最高裁のトランプ関税違憲判決と新関税体制

- **日付**: 2026-02-26
- **カテゴリ**: World
- **ソース**: [CNBC](https://www.cnbc.com/2026/02/20/supreme-court-trump-tariffs-ruling.html)

## 概要

2026年2月20日、米連邦最高裁判所は6対3の判決で、トランプ大統領が国際緊急経済権限法（IEEPA）に基づいて課していた関税を違憲と判断した。ロバーツ首席裁判官が多数意見を執筆し、「IEEPAは大統領に関税を課す権限を付与していない」と明確に判示。これにより従来の10〜50%の関税は即座に無効化された。トランプ大統領は判決の4日後、1974年通商法Section 122を根拠とした新たな10%グローバル関税を発動した。

## 詳細

### 最高裁判決の内容

**事件名**: Learning Resources Inc. v. Trump / V.O.S. Selections v. United States

**判決構成**:
- 多数派（6名）: ロバーツ首席裁判官（主文執筆）、ソトマイヨール、ケーガン、ゴーサッチ、バレット、ジャクソン
- 反対派（3名）: トーマス、アリート、カバノー

**主要な論理**: 憲法上、課税権は議会に帰属する。IEEPAは国際取引の「規制」権限を大統領に付与するものであり、「課税」権限を付与するものではない。トランプ政権の法解釈は「大統領の関税政策権限の変革的拡大」を意味するものであり、議会の意図を逸脱している。

### 経済的インパクト

- **返金義務**: IEEPA関税として徴収済みの関税収入のうち、最大1,750億ドルが輸入業者に返金される可能性
- **歳入影響**: 2026年のIEEPA関税による連邦税収増は推定1,711億ドル（GDP比0.54%）で、1993年以来最大の増税規模だった
- **対象輸入額**: 年間約1.2兆ドル（全輸入の34%）に影響

### 新関税体制（Section 122）

最高裁判決を受けてトランプ大統領が発動した新関税の特徴:

| 項目 | 内容 |
|------|------|
| 法的根拠 | 1974年通商法 Section 122 |
| 税率 | 10%（大統領は15%への引き上げを示唆） |
| 上限税率 | Section 122では最大15%が上限 |
| 期間 | 150日間の時限措置（議会による延長が必要） |
| 適用範囲 | 免除品を除くすべての輸入品 |
| 発効日 | 2026年2月24日 |

### 各国の反応

- **中国**: 一方的な関税の撤廃を要求しつつ、新たな交渉ラウンドの意思を示唆
- **日本**: 既存の通商協定と同等の扱いを求め、昨年の合意を「誠実かつ迅速に実施する」と表明
- **EU**: 15%ベース税率との協定との矛盾について、米国当局から合意遵守の保証を取得
- **業界団体**: 靴類・蒸留酒業界などが市場の予測可能性回復を歓迎

### 民主党の対応

民主党はSection 122関税を150日間の期限を超えて延長するいかなる試みも阻止すると宣言。これにより、関税は2026年7月下旬までに議会の承認がなければ自動的に失効する構図となった。

## 今後の注目点

- **議会での関税延長審議**: 150日の期限切れ（2026年7月下旬予定）に向けた議会の動向。民主党が阻止を表明しており、延長は困難な情勢
- **代替的関税戦略**: トランプ政権は議会を経由せずに関税を課す他の法的根拠を模索中。ベッセント財務長官がIEEPA以外の法律を活用する方針を示唆
- **返金プロセス**: IEEPA関税で徴収された最大1,750億ドルの返金手続きの進行状況
- **国際通商交渉**: 各国との二国間交渉の行方。特に中国との交渉再開の見通し
- **実効関税率の推移**: Deutsche Bankは年内に実効関税率が低下する可能性を予測
