# AI モデルラッシュ — 2026年3月の大量リリース

- **日付**: 2026-03-14
- **カテゴリ**: Tech
- **ソース**: [Sci-Tech Today](https://www.sci-tech-today.com/news/march-2026-ai-models-avalanche/), [LLM Stats](https://llm-stats.com/ai-news), [CNBC](https://www.cnbc.com/2026/02/17/china-alibaba-qwen-ai-agent-latest-model.html)

## 概要

2026年3月第1週（3月1-8日）に、OpenAI、Alibaba、Meta、ByteDance、Tencent、Lightricks など主要プレイヤーと複数の大学が、わずか7日間で12以上の主要 AI モデル・ツールを発表した。言語モデル、動画生成、3D、エージェントなど多岐にわたるリリースラッシュは、AI 業界の競争が新たなフェーズに入ったことを示している。

## 詳細

### OpenAI — GPT-5.4

- 100万トークンのコンテキストウィンドウを実装
- GPT-5.2 比で事実誤認を33%削減
- API 価格は100万入力トークンあたり2.50ドルから
- Standard、Thinking、Pro の3バリアント展開

### Alibaba — Qwen 3.5 Small シリーズ

- 0.8B〜9B パラメータの小型モデルシリーズ
- **9B モデルが13倍の規模（120B）のモデルに匹敵する性能**を達成
  - GPQA Diamond: 81.7（vs GPT-OSS-120B の 71.5）
  - HMMT Feb 2025: 83.2（vs 76.7）
- スマートフォン・ラップトップ上でのオンデバイス動作を実現
- 3月には「AI サービス」を開始し、1コマンドで生活サービスの注文が可能に
- エージェント機能を前面に押し出す戦略

### ByteDance / 北京大学 / Canva — Helios

- 14B パラメータの自己回帰拡散モデル
- 単一の NVIDIA H100 GPU で **19.5 FPS のリアルタイム動画生成**を実現
- 分単位の動画を生成可能
- Apache 2.0 ライセンスでオープンソース公開

### Anthropic — Claude 4.6 Opus

- SWE-bench 75.6% を達成し技術分野のリーダーに
- 1M コンテキストウィンドウ（ベータ）
- 128K 出力トークン

### Google — Gemini 3.1 Pro

- ARC-AGI-2 スコア 77.1% — 前世代から推論性能が2倍以上に向上
- 価格据え置き（$2/$12）での大幅性能向上

### その他の注目リリース

- Lightricks、Tencent、Meta からもモデル・ツールが発表
- 複数の大学研究チームによるオープンソースモデルの公開が加速

## 業界トレンド分析

### 小型モデルの台頭

Qwen 3.5 の9Bモデルが120Bモデルに匹敵する性能を示したことは、「大きいほど良い」というスケーリング則に対する反例となっている。エッジデバイスでの推論が実用段階に入り、クラウド依存からの脱却が進む。

### 動画生成の高速化

Helios の19.5 FPS リアルタイム生成は、動画 AI がバッチ処理からリアルタイム処理へ移行する転換点を示唆している。

### エージェント機能の標準化

OpenAI、Alibaba、Anthropic いずれもエージェント機能を強化しており、「チャット」から「タスク実行」へのパラダイムシフトが加速している。
