# Big Tech capex $725B 時代と AI 収益化の温度差

- **日付**: 2026-05-03
- **カテゴリ**: Tech
- **ソース**: [Tom's Hardware](https://www.tomshardware.com/tech-industry/big-tech/big-techs-ai-spending-plans-reach-725-billion), [Fortune](https://fortune.com/2026/04/29/microsoft-meta-google-ai-capex-spending-billions/), [The Motley Fool](https://www.fool.com/investing/2026/05/01/amazon-alphabet-microsoft-meta-and-apple-just-repo/), [The Next Web](https://thenextweb.com/news/alphabet-amazon-meta-q1-2026-earnings-ai-cloud)

## 概要

2026 年 Q1 決算で Alphabet・Amazon・Meta・Microsoft・Apple の 5 社が出揃った。売上・利益はおおむねコンセンサスを上回ったが、注目点は **5社合計の 2026 年 capex が $650B〜$725B（前年比 +77%）に到達する**規模感と、企業間で AI 投資がきちんと収益化に繋がっているかどうかの差である。

Google Cloud は前年比 +63%、AWS は +28%、Azure 系列も +40% と成長加速の一方で、Meta は capex 上方修正を理由に決算後に売られた。AI バブル懸念と「AI 利益の正体」の議論が並走する 1 週間となった。

### 「AI 利益」の正体問題

Fortune の Eye on AI が問題提起したのは、「Google・Amazon の Q1 ブロックバスター利益のうち、Anthropic への出資評価益の影響を控除すると、本業の AI 収益貢献はどの程度残るのか」という論点である。

- **Amazon**：Anthropic への投資累計 $33B 超。ストック評価で利益が水増しされている可能性。
- **Google (Alphabet)**：最大 $40B の Anthropic 出資。同様に評価益のインパクトが大きい。
- これらを除いた「コア事業ベースの AI 収益化」を見ると、**Google Cloud 単体（+63%）と Microsoft Azure（+40%）は本業ベースで明確な AI ペイオフ**が見える一方、Meta は広告 AI による LTV 改善が capex に追いついていない。

### capex 急騰の構造

- 5社合計 $650B〜$725B のうち、約 6〜7 割が AI/データセンター関連。
- 主要な使途：
  1. Vera Rubin / Blackwell Ultra 系 GPU クラスタ
  2. ハイパースケーラー独自シリコン（Trainium / TPU / Maia / MTIA）
  3. データセンター電源・送電・空冷/液冷インフラ
  4. ファイバー / 海底ケーブル / 衛星リンク
- KKR の Helix Digital Infrastructure（$10B 超）に代表されるように、Big Tech 外部からも AI 専用インフラへの資金流入が加速。

### ベア派の反論と反論への反論

- ベア論：「2026 年 capex は減価償却で 2027〜2028 年の利益を押し下げる」「需要が premium GPU 中心で、汎用クラウドへの溢出は限定的」
- ブル論（Tom's 引用）：「データセンター稼働率は 90% 超で需給逼迫」「inference 需要は学習以上のペースで増えており、減価償却を上回る revenue/GPU が出る」
- 現状の決算ベースでは Google Cloud の +63% と $462B backlog が「需給逼迫が本物である」ことを最も強く示している。

## 含意

### 1. 「全社並走 → 勝者集中」フェーズへ

これまでは「AI に投資すれば株価が上がる」状態だったが、Q1 2026 から **収益化が見える企業（Google・Amazon・Microsoft）と、見えない企業（Meta）の選別**が始まった。capex を出すだけでは評価されない。

### 2. インフラ投資の外部化

KKR Helix のような独立系 AI インフラ事業の登場は、Big Tech が自社 capex を膨らませ続ける負担に対して、リース型/共同所有型のオフバランスシート構造を志向し始めていることを示唆する。

### 3. 円・新興国への波及

Big Tech の調達需要は HBM・先端 NAND・電源・銅・電力契約まで波及しており、世界のサプライチェーンと電力市場に強い拘束力を持ち始めた。日本・台湾・韓国の半導体材料/装置業界への発注は引き続き高水準。
