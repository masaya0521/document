# Drift Protocol $2.85億ハック — 北朝鮮のソーシャルエンジニアリング攻撃の全容

- **日付**: 2026-04-07
- **カテゴリ**: Tech
- **ソース**: [The Hacker News](https://thehackernews.com/2026/04/285-million-drift-hack-traced-to-six.html)

## 概要

2026年4月1日、Solanaブロックチェーン上の分散型永続先物取引所Drift Protocolから約2.85億ドルの暗号資産が流出した。これは2026年最大のDeFiハックであり、Solana史上2番目の規模（2022年のWormholeブリッジハック3.26億ドルに次ぐ）。TRM Labsの調査により、北朝鮮の国家支援ハッカーグループUNC4736（別名: AppleJeus, Citrine Sleet, Lazarus Group関連）による6ヶ月に及ぶ計画的攻撃であることが判明した。

## 詳細

### 攻撃タイムライン

| 時期 | フェーズ |
|------|---------|
| 2025年秋 | ソーシャルエンジニアリングキャンペーン開始。架空の量的取引企業を装い、国際暗号会議でDrift貢献者に接触 |
| 2025年秋〜2026年2月 | 6ヶ月間にわたる信頼構築。詳細な身分・職務経歴・職業ネットワークを作成 |
| 2025年12月〜2026年1月 | エコシステムVaultのオンボーディング段階 |
| 2026年2月〜3月 | 継続的な統合に関する会話 |
| 2026年4月1日 | 攻撃実行。約12分間で資産を流出させ、数時間以内にEthereumへブリッジ |

### 攻撃手法

この攻撃はスマートコントラクトの脆弱性ではなく、複合的なソーシャルエンジニアリングとプロトコル設計の弱点を突いたものだった。

1. **マルチシグ署名者の操作**: 信頼関係を構築した上で、マルチシグ署名者に隠された認可への事前署名を誘導
2. **Security Council移行のゼロタイムロック**: プロトコルの最後の防衛線であるタイムロックを排除
3. **偽トークンの製造**: 「CarbonVote Token」という架空のトークンを作成し、少額の流動性とウォッシュトレードでDriftのオラクルに数億ドル相当の正当な担保として認識させた
4. **推定される初期感染ベクトル**:
   - VS Codeプロジェクト内の悪意ある `tasks.json` ファイル（フォルダ開封時に自動実行）
   - Apple TestFlight経由の偽装ウォレットアプリ

### 北朝鮮サイバー作戦の進化

北朝鮮は近年、サイバー作戦を「意図的に断片化された」マルウェア生態系へと再編している。

- **Kimsuky**: スパイ活動特化
- **Lazarus Group**: 金銭犯罪特化
- **Andariel**: 戦略的破壊特化

この分業体制により帰属回避を強化し、各グループが専門領域に集中できる体制を構築している。

### 被害後の対応

Driftはサービスを一時停止し、被害ユーザーへのエアドロップを提案したが、コミュニティからは反発も出ている。

## 今後の注目点

- **DeFiプロトコルのマルチシグ運用**: ソーシャルエンジニアリングに対する署名プロセスの強化が急務
- **タイムロック設計**: Security Council権限移行にゼロタイムロックを許容する設計の見直し
- **オラクル操作対策**: 新規トークンの担保適格性判定における検証メカニズムの強化
- **北朝鮮の暗号資産攻撃**: 2026年だけで推定10億ドル以上の暗号資産を窃取しており、核・ミサイル開発の資金源として国際的な監視強化が必要
- **被害資金の追跡**: ブリッジされたEthereum上の資金の凍結・回収の可否
