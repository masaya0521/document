# ホルムズ海峡危機 — 史上最大の石油供給途絶とグローバル経済への影響

- **日付**: 2026-03-17
- **カテゴリ**: World
- **ソース**: [Wikipedia](https://en.wikipedia.org/wiki/2026_Strait_of_Hormuz_crisis), [NPR](https://www.npr.org/2026/03/04/nx-s1-5736104/iran-war-oil-trump-israel-strait-hormuz-closed-energy-crisis)

## 概要

2026年2月28日、米国・イスラエルによるイラン攻撃（エピック・フューリー作戦）を契機に、ペルシア湾の要衝ホルムズ海峡がほぼ封鎖状態に陥った。世界の海上石油輸出の約3分の1が通過するこの海峡の機能停止は、1970年代以来最大のエネルギー供給危機を引き起こし、原油価格は126ドルまで急騰。グローバル経済全体に深刻な影響を及ぼしている。

## 詳細

### 時系列

| 日付 | 出来事 |
|------|--------|
| 2月28日 | 米・イスラエルがイランを攻撃。最高指導者ハメネイが殺害される。IRGC が海峡通過禁止を警告 |
| 3月1〜2日 | 複数の船舶が攻撃を受け、乗組員に死傷者 |
| 3月4日 | IRGC が海峡の「完全支配」を宣言 |
| 3月5日 | イラン、米国・イスラエル・西側同盟国の船舶のみ通行禁止と発表。中国・インド等の船舶は選別的に許可 |
| 3月8日 | ブレント原油が4年ぶりに100ドル超。ピーク時126ドルに到達 |
| 3月10日 | イランが機雷敷設を開始。サウジは東西パイプライン経由でヤンブ港への迂回を拡大 |
| 3月11日 | 複数船舶への一斉攻撃。タンカー2隻が炎上 |
| 3月16日 | 海上交通が段階的に再開し始める |

### 被害状況

- 確認された船舶攻撃: 21件
- 損傷した商船: 少なくとも11隻（6隻が放棄）
- タンカー通行量: 初期段階で約70%減少 → その後ほぼゼロ
- 中東全域の死者: 2,200人以上（イラン国内1,300人超）

### エネルギー市場への影響

- **原油価格**: ブレント100ドル超 → ピーク時126ドル
- **天然ガス（欧州）**: €30/MWh → €60/MWh超
- **ガソリン（カリフォルニア）**: 1ガロン5ドル超
- **供給不足**: 日量約1,200万バレル（パイプライン迂回能力の限界）
- **サウジ生産削減**: 日量1,000万 → 800万バレル（20%減）

### 各国の対応

| 国 | 対応 |
|----|------|
| 米国 | タンカー護衛開始を表明。ロシア石油制裁を一時緩和 |
| フランス | 護衛任務「オペレーション・アスピデス」を立ち上げ |
| 英国 | 海峡確保に向けた検討を開始 |
| 日本・オーストラリア | 軍艦派遣を見送り |
| イラン | 選別的通行許可（中国・インド等は許可、西側は禁止） |

### 代替ルート

- **サウジアラビア**: 東西原油パイプライン → 紅海ヤンブ港
- **UAE**: アブダビ原油パイプライン → アラビア海フジャイラ港
- **大手海運**: 喜望峰経由での迂回

ただしパイプライン容量は日量350〜550万バレルで、海峡経由の2,000万バレルには到底及ばない。

## 今後の注目点

- 国際護衛任務の形成と海峡通行の正常化スケジュール
- 原油価格の高止まりが世界経済（特にインフレ・消費）に与える中長期的影響
- イランの選別的通行許可が地政学的パワーバランスに与える影響（中国・インドの立場）
- 米国の戦略石油備蓄（SPR）放出の効果と限界
- エネルギー安全保障の再構築: パイプライン増強・再エネ加速への投資判断
- 日本のエネルギー政策への影響（中東依存度の見直し議論）
