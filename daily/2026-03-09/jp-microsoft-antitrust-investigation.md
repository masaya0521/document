# 公正取引委員会によるマイクロソフト独占禁止法審査

- **日付**: 2026-03-09
- **カテゴリ**: Japan Tech
- **ソース**: [公正取引委員会](https://www.jftc.go.jp/soudan/jyohoteikyo/260304.html)

## 概要

2026年2月25日に公正取引委員会が日本マイクロソフトに立ち入り検査を実施し、3月4日にはマイクロソフト・コーポレーション、日本マイクロソフト、Microsoft Ireland Operations Limited の3社を対象とした独占禁止法違反被疑行為の正式審査を開始した。クラウドサービス市場における競争制限が焦点となっている。

## 詳細

### 疑いの具体的内容

公正取引委員会が問題視しているのは、マイクロソフトのライセンス条件が競合クラウドサービスの利用を制限している可能性である。具体的には以下のソフトウェアが対象とされている：

- Windows Server
- Windows クライアント
- Microsoft SQL Server
- Microsoft 365
- Visual Studio

これらのソフトウェアについて、Azure と競合する他社クラウドサービス（AWS、Google Cloud 等）と組み合わせて利用する場合に、利用を認めない、もしくは Azure 利用者よりも高コストな取引条件を課していた疑いがある。

### 立ち入り検査から正式審査への流れ

- **2月25日**: 公正取引委員会が日本マイクロソフトの東京オフィスに立ち入り検査を実施
- **3月4日**: 正式審査の開始を発表。第三者からの情報・意見の募集を開始
- **5月29日**: 意見募集の締切（午後6時まで）

日本マイクロソフトは「公正取引委員会からの要請に対して全面的に協力する」との姿勢を示している。

### グローバルな独禁法調査の動き

マイクロソフトのクラウド・ライセンス慣行に対する規制当局の監視は世界的に広がっている：

- **米国**: 連邦取引委員会（FTC）がクラウド・AI事業を対象に独禁法調査を強化
- **欧州**: EU競争法に基づく監視・予備調査が進行
- **英国**: 競争・市場庁（CMA）による調査
- **その他**: ブラジル、スイスなどでも同様の動き

日本の公正取引委員会による調査は、こうした世界的な規制強化の流れの中で行われている。

### 日本のクラウド市場への影響

日本のクラウドサービス市場では、AWS、Microsoft Azure、Google Cloud の3大プロバイダーが大きなシェアを占めている。マイクロソフトのライセンス条件が是正された場合、企業のマルチクラウド戦略が促進され、競争環境が改善される可能性がある。

一方で、Microsoft 365 をはじめとするマイクロソフト製品は日本企業の業務基盤として深く浸透しており、ライセンス体系の変更は企業のIT戦略に大きな影響を与えうる。

- 米国FTCや欧州の調査との連動・相互影響
- 日本企業のマルチクラウド移行の加速の有無
- 他のクラウドプロバイダー（AWS、Google Cloud）のライセンス慣行への波及効果
