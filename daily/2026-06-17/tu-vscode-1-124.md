# VS Code — v1.124（2026年6月）

- **日付**: 2026-06-17
- **カテゴリ**: Dev Tool
- **ソース**: [Release Notes v1.124](https://code.visualstudio.com/updates/v1_124), [Releasebot - VS Code June 2026](https://releasebot.io/updates/microsoft/visual-studio-code)

## 概要

VS Code 1.124（2026年6月10日リリース）は、エージェントセッションをまたいだ作業の高速化と、エージェントの自律性向上に主眼を置いたリリース。複数のエージェントセッションを並行管理する「Agents ウィンドウ」、タスク完了判定を賢くした「Autopilot」のデフォルト有効化、統合ブラウザの履歴機能などが目玉となっている。AI エージェントを開発ワークフローの中核に据える方向性がさらに進んだ。

## 主な変更点

### Agents ウィンドウ（プレビュー）

- **バックグラウンドセッション**: `Alt+Enter` でリクエストをバックグラウンド実行し、次のセッションの構成を続けられる
- **セッション間ナビゲーション**: `Ctrl+R` でセッションピッカー、`Ctrl+Tab` で履歴遷移、`Alt+↑/↓` で隣接セッションへ移動。キーボードだけでセッションを検索・ジャンプ・ステップ実行できる
- **リロード時のセッション復元**: レイアウトと状態が自動保持される
- **全セッション一括終了**: `Ctrl+K Ctrl+W` で複数セッションをまとめて閉じる

### Autopilot（プレビュー）

- **デフォルト有効化**: ユーザーの明示的な承認なしに自律的に動作可能になった
- **Advanced Autopilot**: 「タスクが本当に完了したか」の判定にユーティリティモデルを導入。最大3ループまで反復し、より完全な結果を目指す

### 統合ブラウザ

- **履歴機能**: 訪問済みページを記録し、URL バー入力時に候補を提案
- **ツールバーカスタマイズ**: オーバーフローメニューのアクションを常時表示化できる
- **テキスト入力効率化**: `typeInPage` ツールに `submit` パラメータが追加され、入力と確定を単一呼び出しで実行可能

### その他

- シンプルファイルダイアログからフォルダを作成できるように
- auxiliary bar（補助サイドバー）をシェブロンで切り替えて画面幅を最適化
- **エンタープライズ**: Copilot プラグインポリシー管理（実験段階）で、組織レベルでプラグインと marketplace を統一管理

## 破壊的変更・移行ガイド

このリリースで明示された破壊的変更はなし。Autopilot がデフォルト有効になった点は、自律動作を望まない場合は設定で無効化を検討する必要がある。

## 今後の注目点

- Agents ウィンドウと Autopilot はいずれもプレビュー段階であり、安定化に向けた UX・キーバインドの変更が続く可能性
- エージェントの自律性が高まる中で、組織レベルでのポリシー管理（プラグイン制御・コンテンツ除外）の整備がエンタープライズ採用の鍵になる
- 統合ブラウザの強化は、エージェントによるブラウザ操作（E2E 検証・スクレイピング的タスク）を VS Code 内で完結させる流れの一環
