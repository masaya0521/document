# Claude Fable 5 / Mythos 5 の公開とサブスク提供の「期限付き」モデル

- **日付**: 2026-06-17
- **カテゴリ**: Tech
- **ソース**: [Anthropic — Claude Fable 5 and Claude Mythos 5](https://www.anthropic.com/news/claude-fable-5-mythos-5)

## 概要

Anthropic は 2026 年 6 月 9 日、新モデル **Claude Fable 5** を一般公開した。これは従来の Opus 4.8 の上位に位置づけられる「Mythos クラス」初の公開モデルで、ソフトウェアエンジニアリング・知識労働・ビジョン・科学研究など、テストされたほぼ全ての能力ベンチマークで state-of-the-art を記録している。同時に、より高い権限を持つ trusted access 向けに **Claude Mythos 5** も導入された。

特に注目されたのは性能そのものよりも、サブスクリプションでの提供形態だ。Fable 5 は 6/9〜6/22 の期間限定で Pro / Max / Team / Enterprise プランに追加費用なしで含まれるが、6/23 以降はサブスクから外され、利用継続には usage credit への切り替えが必要になる。Anthropic はこれを恒久的な価格決定ではなく capacity 制約に伴う暫定措置と位置づけている。

## 詳細

### ベンチマークと能力

- **SWE-Bench Pro で 80.3%** を記録し、次点モデルを 11 ポイント引き離した。実務的なソフトウェアエンジニアリングタスクでの優位が際立つ。
- 最大 **128k 出力トークン**をサポート。長文生成・大規模リファクタリングなどに対応。
- 安全面では、サイバーセキュリティ・生物・化学・蒸留（distillation）といった高リスク要求をセッションごとに監視する分類器が動作し、該当する要求は Opus 4.8 へルーティングされる。

### 価格体系

- **入力 $10 / 100万トークン、出力 $50 / 100万トークン**。
- プロンプトキャッシュによる入力トークン 90% 割引は引き続き適用。

### サブスクリプションの「6 月 22 日問題」

- 6/9〜6/22：Pro / Max / Team / シート課金 Enterprise プランに**無償で同梱**。
- 6/23：サブスクプランから**除外**。以降は usage credit に切り替えて利用する形になる。
- Anthropic は capacity が回復し次第、標準のサブスク機能として復帰させることを目指すと説明している。

### エコシステムへの波及

同時期に Apple が WWDC で複数 AI を切り替えられる Extensions を発表し、Claude が iPhone の選択肢として初めて組み込まれた。フロンティアモデルの提供チャネルがクラウド API からコンシューマデバイスへと広がる動きの一環といえる。

## 今後の注目点

- 6/23 のサブスク除外が予定どおり実施されるか、capacity 回復による「復帰」がいつ実現するか。
- $50/Mtok という出力価格が、Opus 4.8 からの移行を検討する開発者の採用判断にどう影響するか。
- Mythos クラスの登場で、Opus / Sonnet / Haiku というティア構成がどう再編されていくか。
- 高リスク要求を Opus 4.8 へルーティングする分類器の運用が、実利用での誤検知・遅延にどう作用するか。
