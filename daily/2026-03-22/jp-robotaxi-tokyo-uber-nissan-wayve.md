# Uber・日産・Wayve ロボタクシー東京実証 — 日本の自動運転実装が加速

- **日付**: 2026-03-22
- **カテゴリ**: Japan Tech
- **ソース**: [日産ニュースルーム](https://global.nissannews.com/ja-JP/releases/260312-01-j)、[Impress Watch](https://www.watch.impress.co.jp/docs/news/2092633.html)

## 概要

2026年3月12日、英国AI自動運転スタートアップWayve、米Uber Technologies、日産自動車の3社がロボタクシーの協業に関する覚書（MOU）を締結したと発表した。WayveのエンドツーエンドAI自動運転システム「AI Driver」を搭載した新型「日産リーフ」を使い、2026年後半に東京でロボタクシーの試験運行を開始する。UberにとってJapan初の自動運転パートナーシップとなる。

## 詳細

### 各社の役割分担

| 企業 | 役割 |
|------|------|
| Wayve（英国） | エンドツーエンドAI自動運転システム「AI Driver」の開発・提供 |
| 日産自動車 | ベース車両（新型リーフ）の提供、車両プラットフォーム |
| Uber | ライドヘイリングプラットフォーム、ユーザーとの接続基盤 |

### 技術的特徴

WayveのAI Driverは、従来のルールベースの自動運転システムとは異なり、エンドツーエンドの深層学習モデルを使用する。カメラやセンサーからの入力を直接運転操作に変換するアプローチで、複雑な都市環境への適応力が高いとされる。

### 実証計画

- 開始時期: 2026年後半
- 場所: 東京
- 運行形態: 初期は訓練されたセーフティドライバーが同乗
- プラットフォーム: Uberアプリから利用可能
- Wayveのグローバル展開の一環（ロンドン含む10都市以上に展開予定）

### 日本の自動運転をめぐる動き

東京では同時期に複数の自動運転プロジェクトが進行している:

- **Uber・日産・Wayve**: ロボタクシー（本件）
- **Nuro**: 東京で自動配送ロボットの路上テスト開始（3月11日発表）。Nvidia、トヨタが出資。
- **ホンダ**: Level 4自動運転の実証を推進中
- **国土交通省**: 自動運転レベル4の運行許可制度を整備

### 規制環境

日本政府は2023年の道路交通法改正でレベル4自動運転の運行許可制度を導入しており、規制面での環境は整いつつある。デジタル庁も国際データガバナンス有識者会議でAIシステムのデータアクセス・利用ルールを議論している。

### 海外との比較

米国ではWaymo（Google）やZoox（Amazon）が複数都市でロボタクシーの商用サービスを展開済み。中国ではBaiduのApollo Goが広範囲に運行中。日本は規制面では先進的だが、実装面では後発であり、今回の3社協業は日本市場での自動運転実装を加速させる重要な一歩となる。

## 今後の注目点

- 東京のどのエリアで試験運行が開始されるか（特区指定との関連）
- セーフティドライバーなしの完全無人運行への移行タイムライン
- 東京の複雑な交通環境（狭い道路、自転車、歩行者）でのAI Driverの性能
- Nuroの配送ロボットとの共存・棲み分け
- 日本の消費者のロボタクシーに対する受容度
- 既存のタクシー業界との関係性・規制調整
- Wayveのグローバル展開における東京の位置づけ
