# GPT-5.4 と AI 業界再編の加速

- **日付**: 2026-03-22
- **カテゴリ**: Tech
- **ソース**: [OpenAI](https://openai.com/index/introducing-gpt-5-4/), [TechCrunch](https://techcrunch.com/2026/03/05/openai-launches-gpt-5-4-with-pro-and-thinking-versions/), [OpenAI Astral](https://openai.com/index/openai-to-acquire-astral/)

## 概要

OpenAI が3月5日にリリースした GPT-5.4 は、100万トークンのコンテキストウィンドウと初のネイティブ Computer Use 機能を搭載し、デスクトップ自動化ベンチマーク OSWorld-V で人間の専門家を初めて上回った。さらに3月19日には Python ツール企業 Astral（uv, Ruff, ty の開発元）の買収を発表し、「コード生成」から「ソフトウェア開発ライフサイクル全体への参加」への転換を鮮明にしている。

## 詳細

### GPT-5.4 の技術的進歩

**コンテキストウィンドウ**: API・Codex 版は最大105万トークン（入力922K + 出力128K）をサポート。272Kトークンを超える入力は2倍の料金が適用される。

**Computer Use 機能**: OpenAI 初の汎用モデルとして組み込みの Computer Use 機能を搭載。スクリーンショット、マウスコマンド、キーボード入力を通じてソフトウェアを直接操作できる。

**OSWorld-V ベンチマーク**:
- GPT-5.4: **75.0%**（State-of-the-Art）
- 人間の専門家: 72.4%
- GPT-5.2: 47.3%

これは、実際のデスクトップ環境でのタスク遂行能力において AI が人間を初めて上回ったことを意味する。GPT-5.2 からの飛躍（47.3% → 75.0%）は特に顕著。

**バリエーション**: GPT-5.4 には通常版と Thinking（推論強化）版の2つのバリエーションが提供されている。

### Astral 買収の戦略的意図

**Astral とは**: Python エコシステムで広く使われるオープンソースツール群の開発元。
- **uv**: 高速な Python パッケージマネージャ
- **Ruff**: 超高速な Python リンター・フォーマッター
- **ty**: Python 型チェッカー

これらは数百万の開発者ワークフローで使用されており、モダン Python 開発の基盤となっている。

**OpenAI の狙い**: Codex エコシステムを「コードを生成するだけの AI」から「変更の計画、コードベースの修正、ツールの実行、結果の検証、ソフトウェアのメンテナンス」まで参加できるシステムへ進化させること。Astral のツール群は、Codex がコードの品質を担保しながら自律的に開発を行うための基盤となる。

**Codex の成長**: 2026年初頭からユーザー数3倍、利用量5倍に成長し、週間アクティブユーザー200万人を突破。

**オープンソースへのコミットメント**: 買収後も Astral のオープンソース製品は維持・サポートされる方針。

### AI 業界の再編加速

GPT-5.4 と Astral 買収は、AI 業界全体の動きと連動している。

- **Atlassian**: 従業員1,600人（全体の10%）をレイオフし、AI 開発に資源を集中
- **Google**: Stitch デザインツールに音声「バイブデザイン」機能を追加し、Figma に挑戦
- **DORA Report**: AI は自動的にソフトウェアデリバリーを改善するのではなく、既存の組織力を増幅するものと結論

## 今後の注目点

- **Astral 買収の規制審査**: 独占禁止法の観点で開発者ツール市場への影響が審査される可能性
- **Codex の進化**: Computer Use + Astral ツール統合で、自律的なソフトウェア開発エージェントがどこまで実用化されるか
- **競合の対応**: Anthropic（Claude）、Google（Gemini）が同様の開発者ツール統合をどう進めるか
- **オープンソースコミュニティの反応**: Astral ツールのオープンソース維持が実際に守られるか、コミュニティの監視が続く
- **エンタープライズ AI 導入**: Atlassian のレイオフに象徴される「AI への投資シフト」が他の大手テック企業に波及するか
