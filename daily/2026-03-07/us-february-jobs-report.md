# 米国2月雇用統計の衝撃 — 経済減速シグナルの深掘り

- **日付**: 2026-03-07
- **カテゴリ**: World
- **ソース**: [CNBC](https://www.cnbc.com/2026/03/06/february-2026-jobs-report.html), [BLS](https://www.bls.gov/news.release/empsit.nr0.htm)

## 概要

3月6日に発表された米国の2月雇用統計は、非農業部門雇用者数が9.2万人減少という衝撃的な結果となった。コンセンサス予想の5.6万人増を大幅に下回り、過去5ヶ月で3度目の雇用減少。失業率は4.4%に上昇し、2021年以来の高水準に迫る。イラン紛争による原油高と合わせて、スタグフレーションへの懸念が一気に高まっている。

## 詳細

### 主要指標

| 指標 | 結果 | 予想 | 前月 |
|------|------|------|------|
| 非農業部門雇用者数 | -9.2万人 | +5.6万人 | +12.6万人（下方修正） |
| 失業率 | 4.4% | 4.3% | 4.3% |
| 平均時給（前月比） | +0.4% | +0.3% | - |
| 平均時給（前年比） | +3.8% | +3.7% | - |
| 平均失業期間 | 25.7週 | - | - |

### セクター別の動向

- **医療**: 2.8万人減。Kaiser Permanenteのストライキ（ハワイ・カリフォルニアで3万人以上が参加）が主因
- **連邦政府**: 1万人減。トランプ政権による連邦職員削減の一環で、2024年10月以降累計33万人（全体の11%）が減少
- **エネルギー**: 原油高を背景にExxon Mobil、Chevron、Occidental Petroleumが上昇

### 賃金の逆行

雇用が大幅に減少する中、賃金は予想を上回る伸びを見せた。平均時給は前月比+0.4%、前年比+3.8%と、いずれも予想を0.1ポイント上回った。労働市場のタイト化と物価上昇の複合効果を示唆しており、FRBの金融政策判断を一層複雑にしている。

### 長期失業の悪化

平均失業期間は25.7週に達し、2021年12月以来の最長。一度職を失うと再就職が困難な構造的問題が浮き彫りに。

### 市場への影響

- **ダウ平均**: -453ポイント（-0.9%）→ 47,501.55
- **S&P 500**: -90.69ポイント（-1.3%）→ 6,740.02
- **ナスダック**: -361.31ポイント（-1.6%）→ 22,387.68
- **原油（ブレント）**: 90ドル突破、週間では2022年以来最大の上昇率
- テック株全面安: NVIDIA -1.4%、Microsoft -0.7%、Apple -0.8%、Alphabet -1.2%、Meta -1.1%

## 今後の注目点

- Kaiser Permanenteストライキ終結後の医療セクターの回復と3月雇用統計への反映
- 連邦政府の人員削減ペースと地方経済への波及効果
- FRBの次回FOMC（3月下旬）での金利判断 — 雇用悪化vs賃金インフレの板挟み
- イラン紛争長期化による原油高がインフレを再燃させるリスク
- 「スタグフレーション」シナリオの現実味と企業の設備投資への影響
