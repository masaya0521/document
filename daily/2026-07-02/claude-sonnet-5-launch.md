# Claude Sonnet 5 の一般公開とエージェント性能

- **日付**: 2026-07-02
- **カテゴリ**: Tech
- **ソース**: [Introducing Claude Sonnet 5（Anthropic）](https://www.anthropic.com/news/claude-sonnet-5) / [VentureBeat](https://venturebeat.com/technology/anthropic-launches-claude-sonnet-5-at-a-steep-discount-to-its-top-model-as-the-company-races-toward-a-blockbuster-ipo)

## 概要

Anthropic は 2026 年 6 月 30 日に Claude Sonnet 5 をリリースし、Free/Pro プランのデフォルトモデルに設定した。Max・Team・Enterprise でも利用でき、Claude Code や Claude Platform、`claude-sonnet-5` API 経由でも提供される。今年最大の一般向けモデル投入であり、多くのタスクでフラッグシップの Opus 4.8 に迫る性能を、より低い価格で実現した点が最大の特徴。

エージェント（自律的にツールを操作してタスクを遂行する AI）としての能力を前面に押し出しており、数ヶ月前ならより大型・高価なモデルが必要だった水準の自律実行を、中位モデルで到達させた点が業界的なインパクトになっている。

## 詳細

### 価格

- **導入価格（2026/8/31 まで）**: 入力 $2 / 100万トークン、出力 $10 / 100万トークン
- **標準価格（9/1 以降）**: 入力 $3、出力 $15
- **比較対象の Opus 4.8**: 入力 $5、出力 $25

標準価格でも Opus 4.8 比で入力・出力ともに約 40% 安い。前世代の Sonnet 4.6 と比べても導入期間中はさらに割安で、エージェントを大量にループ実行するユースケースの単価を大きく引き下げる。

### エージェント性能

Sonnet 5 は「これまでで最もエージェント的な Sonnet」と位置づけられる。計画立案、ブラウザやターミナルといったツールの利用、長時間の自律実行に対応し、推論・ツール利用・コーディング・ナレッジワークといったエージェント性能の主要側面で Sonnet 4.6 から大きく改善している。性能は Opus 4.8 に近く、コスト効率で上回る。

### 事業的な文脈

VentureBeat は、フラッグシップに対する大幅ディスカウントでの投入を、Anthropic が大型 IPO に向けて突き進む中での戦略的な動きと位置づけている。エージェント実行の単価低下は、AI エージェントを常時稼働させるプロダクトの経済性を改善し、採用の裾野を広げる狙いがある。同時期に研究向けの Claude Science も展開しており、用途別の製品ラインを拡張している。
