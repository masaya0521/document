# SpaceX IPO — 史上最大級の上場に向けた S-1 の全容

- **日付**: 2026-05-27
- **カテゴリ**: Tech / World
- **ソース**: [The Motley Fool](https://www.fool.com/investing/2026/05/25/the-spacex-ipo-is-coming-these-are-the-3-most-impo/)、[SEC EDGAR](https://www.sec.gov/Archives/edgar/data/0001181412/000162828026036936/spaceexplorationtechnologi.htm)

## 概要

SpaceX は2026年5月20日に SEC へ S-1 を提出し、ティッカー「SPCX」での IPO 準備を正式に開始した。バリュエーション目標は$1.75兆（約260兆円）で、実現すれば史上最大の IPO となる。宇宙・通信・AI を統合した事業構造、Starlink の急成長、そして個人投資家に30%の株式を割り当てるという異例の構成が注目を集めている。

## 詳細

### 財務概要

| 指標 | 数値 |
|------|------|
| 2025年売上高 | $18.7B（前年比33%増） |
| Starlink 売上 | $11.4B（全体の61%） |
| 2025年 GAAP 純損失 | $4.94B |
| 2026年 Q1 売上 | $4.7B |
| 2026年 Q1 純損失 | $4.3B |
| 累積赤字（2026年3月末） | $41.3B |

SpaceX はオペレーションベースでは黒字化しているものの、GAAP ベースでは大幅な赤字が続いている。これは主に株式報酬や設備投資関連の費用によるものと見られる。

### 事業構成

SpaceX は現在、3つの主要事業で構成されている:

1. **ロケット打ち上げ事業**: Falcon 9、Falcon Heavy、Starship による商業・政府向け打ち上げサービス
2. **Starlink**: 衛星インターネットサービス。売上の61%を占め、急成長中
3. **AI 事業**: 近年追加された新事業領域

### IPO の構造

- **引受幹事**: Goldman Sachs をリードレフトに21行が参加
- **個人投資家枠**: フロートの30%を個人投資家に割り当て（テック IPO としては異例の高比率）
- **Elon Musk の持分**: 株式の42%、議決権の85%を保持
- **上場予定日**: 2026年6月12日

### 市場への影響

$1.75兆のバリュエーションが実現すれば、SpaceX は上場直後に世界で最も価値のある企業の一つとなる。Starlink の急成長が評価の中心だが、ロケット打ち上げ事業の安定性とスケーラビリティも投資家の関心を集めている。

## 今後の注目点

- **6月12日の上場日**: 実際の取引開始後の株価推移と需要動向
- **Starlink の成長持続性**: 衛星インターネット市場での競合（Amazon Project Kuiper 等）との競争激化
- **GAAP 赤字の縮小**: オペレーション黒字を GAAP ベースでも実現できるかが中長期の評価を左右
- **議決権構造**: Musk が85%の議決権を維持する構造に対する投資家・規制当局の反応
- **Starship の商業化**: 次世代ロケット Starship の商業運用開始時期とコスト削減効果
