# OpenAI GPT-5.4 — コンピュータ操作を獲得した汎用モデル

- **日付**: 2026-03-28
- **カテゴリ**: Tech
- **ソース**: [OpenAI](https://openai.com/index/introducing-gpt-5-4/), [TechCrunch](https://techcrunch.com/2026/03/05/openai-launches-gpt-5-4-with-pro-and-thinking-versions/), [DataCamp](https://www.datacamp.com/blog/gpt-5-4)

## 概要

OpenAI が2026年3月5日にリリースした GPT-5.4 は、100万トークンのコンテキストウィンドウとネイティブのコンピュータ操作機能を搭載した最新フロンティアモデルである。OSWorld ベンチマークで人間の性能（72.4%）を超える75.0%を達成し、ChatGPT・API・Codex の同時リリースは OpenAI として初めて。

## 詳細

### コンテキストウィンドウ

- API / Codex 版: 最大100万トークン（入力922K + 出力128K）
- ChatGPT 版（GPT-5.4 Thinking）: GPT-5.2 Thinking と同等
- 272K トークン超の入力プロンプトは入力料金が2倍

### ネイティブコンピュータ操作

GPT-5.4 は OpenAI 初のコンピュータ操作機能内蔵汎用モデル。スクリーンショットの認識、マウス操作、キーボード入力を通じてソフトウェアを直接操作できる。これにより、デスクトップ環境での複数ステップのワークフローを自律実行することが可能になった。

### ベンチマーク性能

| ベンチマーク | GPT-5.4 | GPT-5.2 | 人間 |
|-------------|---------|---------|------|
| OSWorld-Verified | 75.0% | 47.3% | 72.4% |

OSWorld-Verified は、デスクトップ環境でスクリーンショットとキーボード/マウス操作を通じてタスクを遂行する能力を測定するベンチマーク。GPT-5.4 は GPT-5.2 から27.7ポイントの大幅改善を実現し、人間の性能を初めて超えた。

### リリース形態

GPT-5.4 は以下の3プラットフォームで同時リリースされた（OpenAI 初の統一トリプルリリース）:

- **ChatGPT**: Pro および Thinking バージョン
- **API**: 開発者向けアクセス
- **Codex**: コーディング特化用途

### OpenAI の事業規模

- 年間収益が250億ドルを突破
- 2026年末までに従業員数を約8,000人に倍増予定
- IPO の初期準備に着手（2026年後半の可能性）
